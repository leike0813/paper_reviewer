# 医学 / 临床 / 公共卫生论文写作指南

## 1. 适用范围

本指南适用于以**人群、患者、临床实践、健康干预、疾病负担、诊断、预后、卫生服务、公共卫生政策、护理实践、药物使用、医学影像、健康经济学和循证医学证据**为核心对象的医学健康类论文。

适用领域包括但不限于：

- 临床医学；
- 流行病学；
- 公共卫生；
- 护理学；
- 药学与药物流行病学；
- 循证医学；
- 健康政策；
- 健康服务研究；
- 医学影像；
- 诊断研究；
- 预后研究；
- 干预研究；
- 医学人工智能临床评价；
- 医疗质量改进；
- 健康经济学评价；
- 质性健康研究；
- 系统综述与 Meta 分析；
- 病例报告与病例系列。

本指南适合辅助以下写作任务：

- 设计和撰写医学论文；
- 判断医学论文研究类型；
- 匹配相应报告规范；
- 检查 Methods 是否足以支撑可信结论；
- 检查偏倚、混杂、失访、缺失数据和外部效度；
- 重构 Results 和 Discussion；
- 判断统计显著性与临床重要性是否被混淆；
- 检查论文结论是否过度外推；
- 生成医学论文审稿式修改建议；
- 辅助回应审稿意见。

医学 / 临床 / 公共卫生论文的可信度来自五个方面的共同支撑：

```text
研究设计
+ 偏倚控制
+ 统计分析
+ 报告透明性
+ 临床 / 公共卫生意义
```

单靠“样本量大”“P 值显著”“模型准确率高”“来自真实世界数据”都不足以构成强医学证据。医学论文必须让读者能够判断：

```text
研究问题是否重要？
研究设计是否匹配问题？
研究对象是否代表目标人群？
暴露、干预、诊断测试或预测因素是否定义清楚？
结局是否临床相关且测量可靠？
偏倚和混杂是否得到控制？
统计分析是否适当？
结果是否具有临床或公共卫生意义？
报告是否透明到足以被审查、复核和应用？
```

医学健康类论文高度依赖研究类型和报告规范。常见报告规范包括：随机试验通常使用 CONSORT，随机试验方案使用 SPIRIT，观察性研究使用 STROBE，诊断准确性研究使用 STARD，病例报告使用 CARE，系统综述使用 PRISMA，健康经济学评价使用 CHEERS，质性研究可参考 COREQ 或 SRQR，预测模型研究可参考 TRIPOD+AI，常规收集健康数据研究可参考 RECORD，医疗质量改进研究可参考 SQUIRE。上述规范的适用范围应根据具体研究类型确认，不能混用或虚构条款。:contentReference[oaicite:0]{index=0}

## 2. 不适用范围

本指南不应机械套用于以下文本或研究。

### 2.1 基础生物医学实验论文

如果研究主要通过细胞实验、动物实验、分子机制、药理实验或材料表征解释生物学机制，而没有以人群、患者、临床结局或公共卫生结局为核心对象，则更适合实验 / 观察型自然科学论文指南。

### 2.2 纯医学科普或临床经验总结

面向公众的健康科普、临床经验文章、专家观点或教学病例讨论，不等同于医学研究论文。  
医学研究论文需要明确研究设计、数据来源、分析方法、伦理审批和证据边界。

### 2.3 单纯医院工作总结

例如“某科室护理管理经验”“某医院信息化建设总结”，如果没有系统研究设计、数据收集、分析框架和可评价结局，则不应包装为研究论文。

### 2.4 临床指南或共识文件

临床实践指南、专家共识、推荐意见有其独立方法学要求，如证据分级、推荐强度、专家共识流程等。系统综述或临床研究证据可为指南服务，但指南本身不是普通 IMRaD 研究论文。

### 2.5 纯算法或医学 AI benchmark 论文

如果论文核心贡献是算法、模型或数据集，主要通过 benchmark 与 baseline 评价性能，则应以计算 / 算法 / 数据驱动型论文指南为主。

但如果论文重点是算法在临床诊断、预后、筛查或真实临床路径中的有效性、安全性、偏倚、公平性和临床可用性，则本指南仍应结合使用。

### 2.6 未经伦理和数据合规说明的人体研究

任何涉及患者、受试者、可识别健康数据、医疗记录、样本或敏感健康信息的研究，都必须根据研究性质、地区法规、机构要求和期刊要求说明伦理审批、知情同意、数据保护或豁免依据。缺少这些说明会严重削弱论文可信度。

## 3. 知识生产逻辑

医学 / 临床 / 公共卫生论文通过以下方式生产可信知识：

```text
临床 / 公共卫生问题
→ 明确研究问题
→ 匹配研究设计
→ 定义研究对象、暴露 / 干预 / 测试 / 预测因素、结局
→ 控制偏倚和混杂
→ 使用适当统计分析
→ 报告效应大小、不确定性和临床意义
→ 讨论适用范围和局限
→ 形成可被临床、政策或证据体系评估的结论
```

### 3.1 医学论文的可信知识来源

| 可信性来源 | 写作中应体现什么 | Agent 判断重点 |
|---|---|---|
| 研究问题 | PICO / PECO / PICOTS 或等价结构 | 问题是否可回答 |
| 研究设计 | RCT、队列、病例对照、横断面、诊断、预后、综述等 | 设计是否匹配问题 |
| 研究对象 | 来源、纳排标准、招募流程、代表性 | 是否有选择偏倚 |
| 暴露 / 干预 / 测试 | 定义、测量、执行、依从性、时间窗口 | 是否可复核 |
| 结局 | 主要结局、次要结局、不良事件、测量方法 | 是否临床相关 |
| 偏倚控制 | 随机化、盲法、匹配、调整、敏感性分析等 | 偏倚是否被识别 |
| 混杂控制 | 先验混杂因素、模型调整、倾向评分、分层等 | 是否过度或不足调整 |
| 统计分析 | 效应量、置信区间、模型、缺失数据处理 | 是否适合设计 |
| 报告透明性 | 方案、注册、流程图、完整结果、报告规范 | 是否可审查 |
| 临床意义 | 最小临床重要差异、绝对风险、NNT、危害 | 是否超越 P 值 |
| 外部效度 | 人群、场景、医疗系统、随访时间 | 能否推广 |

### 3.2 研究类型决定论文逻辑

医学论文不能先写成通用 IMRaD，再事后套报告规范。研究类型从根本上决定写作逻辑。

| 研究类型 | 核心问题 | 关键可信性来源 |
|---|---|---|
| 随机对照试验 | 干预是否有效、安全 | 随机化、分配隐藏、盲法、依从性、预注册、意向治疗 |
| 观察性研究 | 暴露与结局是否相关 / 可能因果 | 选择、测量、混杂控制、时间顺序 |
| 诊断研究 | 测试能否准确识别目标疾病 | 目标疾病、参考标准、盲法解释、连续入组、2×2 表 |
| 预后研究 | 哪些因素预测未来结局 | 起点、随访、预测因子、模型验证、校准 |
| 病例报告 | 罕见、异常或教育性病例说明什么 | 时间线、诊疗过程、替代解释、患者视角 |
| 系统综述 | 现有证据总体说明什么 | 检索、筛选、偏倚评价、证据综合、异质性 |
| 公共卫生研究 | 人群层面风险、干预或政策影响 | 人群代表性、测量、生态 / 个体层次、政策背景 |
| 健康经济学评价 | 成本与效果是否值得 | 视角、时间范围、成本、效用、模型、不确定性 |
| 质性健康研究 | 患者、医务人员或系统经验如何被理解 | 采样、访谈、编码、反身性、饱和度、可信度 |

### 3.3 医学研究的 Claim-Evidence-Warrant

医学论文中的每个重要主张都应包含：

```text
Claim：作者声称什么？
Evidence：哪项结果支持？
Warrant：为什么该研究设计和分析可以支持这个主张？
Boundary：适用于哪些人群、场景和时间范围？
```

示例：

```text
Claim：干预 A 降低 2 型糖尿病患者 12 个月 HbA1c。
Evidence：随机试验中，干预组 HbA1c 较对照组平均降低 0.5%，95% CI 为 0.2% 至 0.8%。
Warrant：随机分配和预设主要结局降低了混杂和选择偏倚风险；分析按意向治疗原则进行。
Boundary：结论主要适用于本文纳入的中年门诊患者，不能直接推广到妊娠糖尿病或严重肾功能不全患者。
```

### 3.4 统计显著性不等于临床重要性

医学论文必须区分：

| 概念 | 含义 | 写作要求 |
|---|---|---|
| 统计显著性 | 观察结果与零效应不一致的统计证据 | 报告 P 值和置信区间 |
| 临床重要性 | 效应大小是否足以影响患者或实践 | 报告绝对差异、MCID、NNT / NNH、风险收益 |
| 公共卫生重要性 | 在群体层面是否有实质影响 | 报告人群归因风险、覆盖率、成本、可实施性 |
| 证据确定性 | 结果接近真实效应的信心 | 讨论偏倚、不精确、间接性、不一致 |

弱写法：

```text
干预组显著优于对照组，因此具有临床价值。
```

强写法：

```text
干预组在主要结局上显示统计学差异，但组间差异低于预设最小临床重要差异。因此，该结果的临床意义有限，需要结合成本、不良事件和患者偏好解释。
```

## 4. 典型论文结构

医学论文通常采用 IMRaD 结构：

```text
Title
Abstract
Keywords

1. Introduction
2. Methods
3. Results
4. Discussion
5. Conclusion

Ethics Approval
Funding
Conflicts of Interest
Data Availability
Trial Registration / Protocol, if applicable
References
Supplementary Materials
```

### 4.1 IMRaD 在医学论文中的功能

| 部分 | 核心问题 | 功能 |
|---|---|---|
| Introduction | 为什么研究？ | 疾病负担、临床问题、证据缺口、研究目的 |
| Methods | 如何研究？ | 设计、对象、干预 / 暴露、结局、样本量、统计、伦理 |
| Results | 发现什么？ | 研究流程、基线特征、主要 / 次要结局、不良事件、分析 |
| Discussion | 结果意味着什么？ | 临床意义、证据比较、机制、偏倚、局限、实践影响 |
| Conclusion | 能得出什么结论？ | 回答研究问题，限定适用范围 |

### 4.2 医学论文常见结构变体

#### 随机对照试验

```text
1. Introduction
2. Methods
   - Trial design
   - Participants
   - Randomization and masking
   - Interventions
   - Outcomes
   - Sample size
   - Statistical analysis
   - Ethics and registration
3. Results
   - Participant flow
   - Baseline characteristics
   - Primary outcome
   - Secondary outcomes
   - Harms
4. Discussion
5. Conclusion
```

随机试验结果报告应参考 CONSORT；试验方案报告可参考 SPIRIT；干预细节报告可结合 TIDieR。CONSORT 2025 包含随机试验报告的 30 项清单和参与者流程图，SPIRIT 2025 用于随机试验方案报告，TIDieR 用于提高干预描述和可复制性。:contentReference[oaicite:1]{index=1}

#### 观察性研究

```text
1. Introduction
2. Methods
   - Study design and setting
   - Participants
   - Variables
   - Data sources / measurement
   - Bias
   - Study size
   - Statistical methods
3. Results
   - Participants
   - Descriptive data
   - Outcome data
   - Main results
   - Sensitivity / subgroup analyses
4. Discussion
```

队列、病例对照和横断面研究通常参考 STROBE；使用电子健康记录、医保数据库、登记系统等常规收集健康数据的观察性研究可结合 RECORD。:contentReference[oaicite:2]{index=2}

#### 诊断准确性研究

```text
1. Introduction
2. Methods
   - Participants
   - Index test
   - Reference standard
   - Target condition
   - Test interpretation
   - Statistical methods
3. Results
   - Participant flow
   - Test results
   - Diagnostic accuracy
   - Indeterminate / missing results
4. Discussion
```

诊断准确性研究应参考 STARD 2015；医学 AI 诊断准确性研究还需关注 AI 特有的透明性、数据划分、外部验证和临床使用场景。STARD 2015 是用于诊断准确性研究透明报告的清单。:contentReference[oaicite:3]{index=3}

#### 预后 / 预测模型研究

```text
1. Introduction
2. Methods
   - Source of data
   - Participants
   - Outcome
   - Predictors
   - Sample size
   - Missing data
   - Model development / validation
   - Performance measures
3. Results
   - Participants
   - Model specification
   - Discrimination
   - Calibration
   - Clinical utility
4. Discussion
```

预测模型研究，包括回归模型和机器学习模型，可参考 TRIPOD+AI；偏倚和适用性评价可考虑 PROBAST 或 PROBAST+AI，但这些是偏倚评价工具而非普通报告清单。:contentReference[oaicite:4]{index=4}

#### 系统综述与 Meta 分析

```text
1. Introduction
2. Methods
   - Protocol and registration
   - Eligibility criteria
   - Information sources
   - Search strategy
   - Selection process
   - Data collection
   - Risk of bias assessment
   - Synthesis methods
3. Results
   - Study selection
   - Study characteristics
   - Risk of bias
   - Results of individual studies
   - Synthesis results
   - Certainty of evidence
4. Discussion
```

系统综述和 Meta 分析通常参考 PRISMA 2020；证据确定性可使用 GRADE 方法进行评价和呈现。PRISMA 2020 包括清单、扩展清单、摘要清单和流程图；GRADE 是用于系统综述和其他证据综合中评价证据确定性的系统方法。:contentReference[oaicite:5]{index=5}

## 5. 各章节写作指南

## 5.1 标题 Title

### 写作目标

医学论文标题应准确表达：

```text
研究对象
+ 疾病 / 健康问题
+ 研究设计
+ 暴露 / 干预 / 测试 / 结局
+ 场景或人群
```

### 推荐标题要素

| 要素 | 示例 |
|---|---|
| 研究设计 | randomized trial, cohort study, diagnostic accuracy study |
| 人群 | adults with chronic kidney disease |
| 暴露 / 干预 | sodium-glucose cotransporter-2 inhibitors |
| 结局 | hospitalization, mortality, quality of life |
| 场景 | primary care, ICU, community setting |
| 时间 | 12-month follow-up |

### 常见标题类型

```text
A randomized controlled trial of X for Y in Z patients
Association between X and Y: a population-based cohort study
Diagnostic accuracy of X for detecting Y in Z setting
Development and validation of a prediction model for X
Cost-effectiveness of X compared with Y for Z
Patient experiences of X: a qualitative study
```

### 常见错误

- 标题不说明研究设计；
- 标题声称因果，但研究只是观察性关联；
- 标题使用“有效”“安全”“优越”，但正文证据不足；
- 标题过度强调统计显著；
- 研究对象不清；
- 结局不清；
- 病例报告未标明病例报告类型。

### Agent 质量判断

```text
1. 标题是否说明研究设计？
2. 是否说明人群和健康问题？
3. 是否避免把关联写成因果？
4. 是否避免夸大疗效、安全性或准确性？
5. 是否与报告规范要求相符？
```

## 5.2 摘要 Abstract

### 写作目标

医学摘要应让读者快速判断研究可信度和临床意义。很多医学期刊要求结构化摘要，常见小标题包括：

```text
Background
Objective
Design
Setting
Participants
Interventions / Exposure
Main Outcomes and Measures
Results
Conclusions
Trial Registration
```

具体格式应遵循目标期刊要求。

### 摘要应包含

- 疾病或公共卫生问题；
- 研究目的；
- 研究设计；
- 研究地点和时间；
- 研究对象；
- 干预、暴露、测试或预测因素；
- 主要结局；
- 样本量；
- 主要结果，含效应量和置信区间；
- 不良事件，如适用；
- 结论和适用边界；
- 注册号，如临床试验适用。

### 好摘要示例骨架

```text
Background: X 疾病造成显著疾病负担，但 Y 干预在 Z 人群中的效果仍不确定。
Objective: 评估 Y 干预对主要结局 A 的影响。
Design: 多中心、随机、双盲、安慰剂对照试验。
Participants: 纳入……，排除……。
Intervention: ……
Main Outcomes and Measures: 主要结局为……
Results: 共随机分配 N 名患者。主要结局在干预组和对照组分别为……，效应量为……，95% CI 为……。不良事件为……
Conclusions: 在本文研究人群中，Y 干预……。结果应限于……情境解释。
```

### 常见错误

- 摘要不写研究设计；
- 只写 P 值，不写效应量和置信区间；
- 不写样本量；
- 不写主要结局；
- 只报告显著结果；
- 不写不良事件；
- 观察性研究结论写成因果；
- 结论不限定研究人群和场景；
- 摘要与正文结果不一致。

### Agent 质量判断

```text
1. 摘要是否说明研究设计？
2. 是否说明研究对象和样本量？
3. 是否报告主要结局和效应量？
4. 是否报告不确定性？
5. 是否说明临床意义或边界？
6. 是否避免因果过度声称？
```

## 5.3 Introduction 引言

### 写作目标

医学论文 Introduction 应从疾病负担、临床或公共卫生问题、已有证据不足引出研究目的。

推荐结构：

```text
疾病 / 健康负担
→ 当前临床 / 公共卫生决策问题
→ 已有证据和不足
→ 本研究目的 / 假设
```

### 应包含内容

1. 疾病或健康问题的重要性；
2. 当前实践或政策中存在的不确定性；
3. 已有研究做了什么；
4. 证据缺口是什么；
5. 为什么该缺口会影响临床或公共卫生决策；
6. 本文研究目的；
7. 必要时给出假设。

### 疾病负担写法

弱写法：

```text
糖尿病是一种常见疾病，严重危害健康。
```

强写法：

```text
2 型糖尿病与心血管事件、肾功能下降和医疗费用增加相关，因此降低长期并发症风险是临床管理的重要目标。
```

### 证据缺口写法

弱写法：

```text
目前相关研究较少。
```

强写法：

```text
尽管既往随机试验显示该干预可改善短期血糖控制，但多数研究排除了老年多病共存患者，且随访时间不足以评估住院和严重低血糖等临床结局。因此，其在真实门诊老年患者中的风险收益仍不明确。
```

### 研究目的写法

```text
本研究旨在评估……
```

```text
我们开展了一项……研究，以检验……
```

```text
本研究的主要目的是比较……对……的影响；次要目的是评估……
```

### 常见错误

- 背景过长，研究问题出现太晚；
- 只写疾病严重，不写决策不确定性；
- 缺口只是“研究少”，没有说明为什么影响临床；
- 不说明研究设计；
- 研究目的与 Methods 不一致；
- 引言中暗示因果，但研究设计不能支持；
- 夸大公共卫生意义。

### Agent 质量判断

```text
1. 是否说明疾病负担或临床问题？
2. 是否说明现有证据不足？
3. 缺口是否与研究设计对应？
4. 是否明确研究目的？
5. 是否避免过度铺陈背景？
6. 是否能自然导向 Methods？
```

## 5.4 Methods 方法

### 写作目标

医学论文 Methods 是可信度核心。读者应能够判断：

```text
研究设计是否能回答问题？
研究对象是否合适？
暴露 / 干预 / 测试 / 结局是否定义清楚？
偏倚和混杂如何控制？
样本量是否充分？
统计分析是否适当？
伦理和报告是否透明？
```

### Methods 基本结构

```text
Study design and setting
Participants
Eligibility criteria
Exposure / Intervention / Index test / Predictors
Comparator
Outcomes
Sample size
Data collection and measurement
Bias control
Statistical analysis
Missing data
Subgroup and sensitivity analyses
Ethics approval and consent
Registration / protocol / reporting guideline
```

### 研究设计

必须说明：

- 研究类型；
- 单中心或多中心；
- 前瞻性或回顾性；
- 研究时间；
- 研究地点；
- 数据来源；
- 随访长度；
- 是否注册或有方案；
- 是否符合某报告规范。

弱写法：

```text
本研究分析我院患者资料。
```

强写法：

```text
本研究为单中心回顾性队列研究，纳入 2018 年 1 月至 2023 年 12 月在某三级医院住院的……患者，随访至……。研究报告参考 STROBE。
```

### 研究对象与纳排标准

应说明：

- 来源人群；
- 招募方式；
- 纳入标准；
- 排除标准；
- 分组方式；
- 筛选流程；
- 样本量；
- 失访情况；
- 代表性。

Agent 应检查：

```text
纳排标准是否与研究问题一致？
排除标准是否可能引入选择偏倚？
是否说明从多少人筛到多少人？
是否有流程图？
```

### 干预 Intervention

适用于随机试验、干预研究、质量改进等。应说明：

- 干预内容；
- 给予者；
- 剂量 / 强度 / 频率；
- 持续时间；
- 交付方式；
- 场景；
- 依从性；
- 干预调整；
- 对照条件；
- 共干预；
- 方案偏离。

干预细节可结合 TIDieR 清单，以提高干预可复制性。:contentReference[oaicite:6]{index=6}

### 暴露 Exposure

适用于观察性研究。应说明：

- 暴露定义；
- 测量来源；
- 暴露时间窗口；
- 暴露分组；
- 剂量或水平；
- 是否随时间变化；
- 测量误差；
- 暴露发生是否早于结局。

### 诊断测试 Index Test 与参考标准 Reference Standard

适用于诊断研究。应说明：

- 目标疾病；
- 指标测试；
- 参考标准；
- 测试阈值；
- 测试执行人员；
- 结果解释是否盲法；
- 测试间隔；
- 不确定或无法解释结果；
- 2×2 表数据。

### 结局 Outcomes

应区分：

- 主要结局；
- 次要结局；
- 安全性结局；
- 探索性结局；
- 患者报告结局；
- 替代结局；
- 复合结局；
- 时间到事件结局。

每个结局应说明：

```text
定义
测量工具
测量时间点
事件判定方式
是否盲法
临床意义
```

### 样本量 Sample Size

应说明：

- 样本量估算依据；
- 主要结局；
- 预期效应大小；
- 显著性水平；
- 把握度；
- 失访率；
- 设计效应，如集群设计；
- 事件数，如回归或预测模型；
- 若为便利样本，应说明并承认限制。

### 统计方法

应说明：

- 描述性统计；
- 主要分析模型；
- 效应量；
- 置信区间；
- 协变量选择；
- 混杂控制方法；
- 缺失数据处理；
- 亚组分析；
- 敏感性分析；
- 多重比较处理；
- 统计软件；
- 显著性阈值；
- 预设和探索性分析的区别。

### 缺失数据

应说明：

- 缺失比例；
- 缺失变量；
- 缺失机制假设；
- 完整病例分析、插补或模型处理；
- 敏感性分析；
- 失访对结果的影响。

### 偏倚和混杂控制

Methods 中应主动说明：

- 选择偏倚；
- 信息偏倚；
- 测量偏倚；
- 观察者偏倚；
- 混杂；
- immortal time bias；
- lead-time bias；
- verification bias；
- attrition bias；
- publication bias，如系统综述；
- confounding by indication，如药物流行病学。

### 伦理审批与注册

应按研究类型说明：

- 伦理委员会批准；
- 知情同意或豁免；
- 数据匿名化和隐私保护；
- 临床试验注册号；
- 方案可用性；
- 资金来源；
- 利益冲突。

ICMJE 推荐临床试验公开、前瞻性注册，并建议在摘要末尾报告注册号；具体要求还需遵循目标期刊和所在地法规。:contentReference[oaicite:7]{index=7}

### 常见错误

- 研究设计不清；
- 纳排标准不明确；
- 暴露和结局定义模糊；
- 主要结局和次要结局未区分；
- 样本量没有依据；
- 统计模型没有说明协变量选择；
- 不处理缺失数据；
- 不说明伦理审批；
- 不说明试验注册；
- 使用真实世界数据但不说明数据质量和算法定义；
- Methods 写得无法复现。

### Agent 质量判断

```text
1. 研究设计是否明确？
2. 是否说明研究对象来源和纳排标准？
3. 暴露 / 干预 / 测试 / 预测因素是否可复核？
4. 结局是否临床相关且定义清楚？
5. 是否说明样本量？
6. 是否说明偏倚和混杂控制？
7. 统计方法是否匹配设计和数据？
8. 是否说明缺失数据？
9. 是否说明伦理、注册和报告规范？
```

## 5.5 Results 结果

### 写作目标

Results 应透明呈现研究流程、样本特征和预设分析结果，不应选择性报告有利结果。

医学 Results 通常按以下顺序：

```text
研究对象流程
→ 基线特征
→ 主要结局
→ 次要结局
→ 安全性 / 不良事件
→ 亚组分析
→ 敏感性分析
→ 补充分析
```

### 研究流程

应报告：

- 筛选人数；
- 纳入人数；
- 分组人数；
- 随访人数；
- 失访人数；
- 排除原因；
- 分析集人数；
- 流程图。

RCT 常用 CONSORT flow diagram；系统综述使用 PRISMA flow diagram；诊断研究也需要清楚呈现参与者流动。:contentReference[oaicite:8]{index=8}

### 基线特征

应报告：

- 人口学信息；
- 疾病严重程度；
- 关键风险因素；
- 重要共病；
- 基线结局水平；
- 分组平衡情况；
- 缺失情况。

注意：

```text
基线表不应只看 P 值判断平衡。
随机试验中，基线差异通常不应被解释为随机化失败，除非差异具有实质意义或提示执行问题。
观察性研究中，基线差异是混杂风险的重要线索。
```

### 主要结局

应优先报告预设主要结局：

- 每组事件数或均值；
- 效应量；
- 绝对差异；
- 相对效应；
- 95% 置信区间；
- P 值，如适用；
- 随访时间；
- 分析集。

常见效应量：

| 结局类型 | 常见效应量 |
|---|---|
| 二分类结局 | 风险差、风险比、比值比 |
| 时间到事件 | 风险比、Kaplan-Meier 曲线、绝对风险 |
| 连续结局 | 均值差、标准化均值差 |
| 计数结局 | 率比 |
| 诊断研究 | 敏感度、特异度、似然比、AUC |
| 预测模型 | C-statistic、校准斜率、校准曲线、决策曲线 |
| 经济评价 | ICER、净货币收益、成本效果可接受曲线 |

### 次要结局

次要结局应明确为次要，避免过度解释。  
如果主要结局不显著而次要结局显著，需要谨慎讨论选择性解释和多重比较问题。

### 不良事件 Harms

医学干预论文必须报告安全性：

- 总不良事件；
- 严重不良事件；
- 特定关注事件；
- 停药 / 停止干预原因；
- 死亡；
- 与干预相关性；
- 事件严重程度。

弱写法：

```text
未观察到明显不良反应。
```

强写法：

```text
干预组和对照组分别有 18/120 与 14/118 名参与者发生至少一次不良事件；严重不良事件分别为 3 例和 2 例。研究样本量不足以排除罕见不良事件风险。
```

### 亚组分析

应说明：

- 是否预设；
- 亚组定义；
- 交互作用检验；
- 样本量是否足够；
- 结果是否探索性；
- 是否避免只看各亚组 P 值。

### 敏感性分析

应说明：

- 改变模型；
- 改变暴露定义；
- 改变结局定义；
- 不同缺失数据假设；
- 排除高偏倚样本；
- 倾向评分方法替代；
- 未测量混杂敏感性分析；
- 负对照分析，如适用。

### 常见错误

- 只报告显著结果；
- 不报告基线特征；
- 不报告失访；
- 不报告主要结局的完整效应量；
- 只写 P 值；
- 不报告不良事件；
- 亚组分析被写成确定结论；
- 敏感性分析结果与主分析不一致但不解释；
- 表格太多但没有结果主线；
- Results 中提前写临床建议。

### Agent 质量判断

```text
1. 是否先报告研究流程和样本量？
2. 是否报告基线特征？
3. 是否优先报告预设主要结局？
4. 是否报告效应量和置信区间？
5. 是否报告绝对效应和临床意义所需信息？
6. 是否报告不良事件？
7. 亚组和敏感性分析是否标明预设或探索性？
8. 是否避免选择性报告？
```

## 5.6 Discussion 讨论

### 写作目标

Discussion 应把结果放回临床、公共卫生和已有证据体系中解释。它不是重复 Results，而是回答：

```text
主要发现是什么？
这些发现临床上重要吗？
与已有研究一致还是冲突？
可能机制是什么？
偏倚和限制如何影响解释？
结果能推广到哪些人群？
对临床实践、公共卫生或政策有什么影响？
```

### 推荐结构

```text
1. 简要总结主要发现；
2. 解释临床 / 公共卫生意义；
3. 与已有证据比较；
4. 讨论可能机制；
5. 讨论偏倚、混杂、失访、缺失数据和限制；
6. 讨论外部效度；
7. 说明实践、政策或研究意义；
8. 谨慎结论。
```

### 临床意义写法

弱写法：

```text
结果具有统计学意义，因此说明该治疗有效。
```

强写法：

```text
虽然干预组主要结局改善达到统计学显著性，但绝对改善幅度较小，且置信区间下限接近无临床意义范围。因此，该干预是否值得常规使用，还需结合患者负担、成本和不良事件解释。
```

### 与已有证据比较

应说明：

- 本结果与哪些研究一致；
- 与哪些研究不同；
- 差异可能来自人群、干预强度、结局定义、随访时间、统计调整或医疗系统；
- 本研究相对已有证据增加了什么。

### 机制解释

机制解释应谨慎。临床研究多数不能直接证明生物机制。

可写：

```text
这一结果可能与……机制一致。
```

不宜写：

```text
本研究证明了……机制。
```

除非研究设计和数据确实直接验证机制。

### 局限性

应具体讨论：

- 选择偏倚；
- 信息偏倚；
- 残余混杂；
- 样本量和不精确性；
- 失访；
- 缺失数据；
- 单中心；
- 随访时间短；
- 替代结局；
- 未测量变量；
- 统计多重比较；
- 外部效度；
- 数据质量；
- 研究设计不能支持因果。

### 实践影响

医学论文应避免在证据不足时直接给出强临床建议。

弱写法：

```text
建议临床广泛推广该方案。
```

强写法：

```text
这些结果提示该方案可能适合……患者，但在纳入常规实践前仍需更大规模、长期随访研究评估安全性和成本效果。
```

### 常见错误

- Discussion 重复结果；
- 将统计显著写成临床有效；
- 观察性研究强行因果解释；
- 不讨论偏倚和混杂；
- 不讨论外部效度；
- 忽略不良事件；
- 只引用支持性研究；
- 结论写成指南推荐；
- 对亚组分析过度解释。

### Agent 质量判断

```text
1. Discussion 是否解释主要发现？
2. 是否讨论临床意义而非仅统计显著？
3. 是否与已有证据比较？
4. 是否具体讨论偏倚和限制？
5. 是否说明外部效度？
6. 是否避免过度临床建议？
7. 是否与研究设计支持的结论强度一致？
```

## 5.7 Conclusion 结论

### 写作目标

Conclusion 应简洁回答研究问题，并限定结论适用范围。

### 推荐结构

```text
在……人群 / 场景中，
本研究发现……
该结果提示……
但由于……限制，
结论应限于……
未来研究需要……
```

### 常见错误

- 结论不说明人群；
- 观察性研究写成因果；
- 诊断准确性研究直接写成改善临床结局；
- 统计显著写成临床推荐；
- 小样本研究写成确定性结论；
- 未提局限；
- 引入新结果。

### Agent 质量判断

```text
1. 是否回答研究目的？
2. 是否包含人群和场景限定？
3. 是否与研究设计匹配？
4. 是否避免过度外推？
5. 是否说明需要进一步研究的具体方向？
```

## 6. 常见贡献类型

医学 / 临床 / 公共卫生论文的贡献通常包括以下类型。

### 6.1 疗效或干预效果贡献

通过随机试验或严谨准实验设计评估干预对临床或公共卫生结局的影响。

质量要求：

- 明确干预和对照；
- 随机化或合理替代设计；
- 主要结局预设；
- 样本量充分；
- 安全性报告；
- 临床意义解释。

### 6.2 风险因素或病因线索贡献

通过观察性研究识别暴露与结局之间的关联，或在一定条件下支持因果推断。

质量要求：

- 暴露早于结局；
- 混杂控制充分；
- 敏感性分析；
- 避免过度因果声称；
- 讨论残余混杂。

### 6.3 诊断价值贡献

评估诊断测试、影像标志物、实验室指标或算法对目标疾病的识别能力。

质量要求：

- 明确目标疾病；
- 合适参考标准；
- 连续或代表性样本；
- 盲法解释；
- 完整报告敏感度、特异度、似然比、预测值；
- 讨论临床路径和预期用途。

### 6.4 预后或预测模型贡献

开发、验证或更新预测模型。

质量要求：

- 明确预测时间点和预测窗口；
- 候选预测因子合理；
- 样本量和事件数充分；
- 内部 / 外部验证；
- 区分 discrimination 与 calibration；
- 报告模型完整公式或可用实现；
- 讨论临床实用性。

### 6.5 疾病负担或流行病学描述贡献

描述患病率、发病率、死亡率、负担、趋势或空间分布。

质量要求：

- 人群代表性；
- 定义清楚；
- 测量方法一致；
- 年龄标准化或适当调整；
- 不确定性报告；
- 讨论监测偏倚。

### 6.6 卫生服务或质量改进贡献

评估医疗服务流程、质量、安全性、可及性或效率改进。

质量要求：

- 明确系统层面问题；
- 干预逻辑清楚；
- 上下文充分；
- 使用过程和结果指标；
- 处理 secular trend；
- 报告可实施性和可持续性。

质量改进报告可参考 SQUIRE 2.0，该规范用于报告系统层面提升医疗质量、安全和价值的工作。:contentReference[oaicite:9]{index=9}

### 6.7 健康经济学贡献

比较不同干预或策略的成本和效果。

质量要求：

- 评价视角；
- 时间范围；
- 成本类别；
- 效果或效用指标；
- 折现；
- 模型假设；
- 不确定性分析；
- 成本效果可接受性；
- 公平性和预算影响，如适用。

### 6.8 质性解释贡献

解释患者、医务人员、照护者或政策执行者的经验、意义和行为机制。

质量要求：

- 采样策略；
- 访谈或观察方法；
- 研究者反身性；
- 编码流程；
- 主题形成过程；
- 数据饱和或信息充分性；
- 代表性引文；
- 可信度策略。

## 7. 证据与论证标准

## 7.1 强医学证据的特征

强医学证据并非只来自某一种设计，而是取决于研究问题与设计是否匹配。

```text
1. 研究问题清楚；
2. 设计适合问题；
3. 人群来源明确；
4. 暴露 / 干预 / 测试 / 结局定义清楚；
5. 偏倚风险低或被充分处理；
6. 混杂控制合理；
7. 样本量和事件数足够；
8. 统计方法透明；
9. 缺失数据和失访处理合理；
10. 效应量、不确定性和临床意义完整报告；
11. 结果与报告规范一致；
12. 结论限于数据支持范围。
```

### 7.2 弱医学证据的特征

弱证据常表现为：

- 研究设计与问题不匹配；
- 纳排标准不清；
- 样本来源不明；
- 只报告 P 值；
- 不报告效应量和置信区间；
- 结局不是临床相关结局；
- 暴露和结局同时测量却写成因果；
- 未控制关键混杂；
- 失访严重但未处理；
- 缺失数据未说明；
- 主要结局事后改变；
- 亚组分析过度解释；
- 不良事件缺失；
- 伦理审批缺失；
- 结论推广到未研究人群。

### 7.3 偏倚类型与写作处理

| 偏倚类型 | 常见于 | 写作中应说明 |
|---|---|---|
| 选择偏倚 | 观察性研究、诊断研究 | 招募流程、纳排标准、参与者流程 |
| 混杂 | 暴露研究、药物流行病学 | 混杂因素选择、调整方法、残余混杂 |
| 信息偏倚 | 暴露或结局测量 | 测量工具、盲法、数据质量 |
| 回忆偏倚 | 病例对照、问卷 | 数据收集方式和限制 |
| 失访偏倚 | 队列、试验 | 失访比例、原因、敏感性分析 |
| 验证偏倚 | 诊断研究 | 谁接受参考标准，是否一致 |
| lead-time bias | 筛查研究 | 结局选择、随访解释 |
| immortal time bias | 药物和数据库研究 | 暴露时间定义和时间依赖分析 |
| 报告偏倚 | 系统综述、试验 | 方案、注册、完整结局报告 |

### 7.4 混杂控制标准

Agent 应检查：

```text
1. 作者是否先验定义混杂因素？
2. 混杂因素是否发生在暴露前？
3. 是否避免调整中介变量？
4. 是否避免调整碰撞变量？
5. 是否说明协变量选择依据？
6. 是否进行敏感性分析？
7. 是否承认未测量混杂？
```

### 7.5 统计显著性与临床重要性

医学论文应报告：

- 效应量；
- 95% 置信区间；
- 绝对风险差；
- 相对风险；
- baseline risk；
- NNT / NNH，如适用；
- MCID，如适用；
- 不良事件；
- 成本和可实施性，如适用。

Agent 判断规则：

```text
如果结论只依据 P < 0.05，而没有讨论效应大小和临床意义，则论证不足。
```

### 7.6 常用偏倚评价工具与证据确定性

系统综述中可根据研究类型选择风险偏倚工具。例如，RoB 2 常用于随机试验偏倚风险评价，ROBINS-I 用于非随机干预研究，QUADAS-2 用于诊断准确性研究，PROBAST 用于诊断或预后预测模型研究。GRADE 常用于系统综述和证据综合中评价证据确定性。这些工具不是普通论文报告规范，Agent 应避免把“报告清单”和“偏倚评价工具”混为一谈。:contentReference[oaicite:10]{index=10}

## 8. 文献综述写法

### 8.1 医学论文文献综述的功能

医学论文中的文献综述通常嵌入 Introduction 和 Discussion。其功能是建立：

```text
疾病 / 健康问题
→ 当前临床或公共卫生决策不确定性
→ 已有证据
→ 证据缺口
→ 本研究如何补充
```

### 8.2 不同研究类型的文献综述重点

| 研究类型 | 文献综述重点 |
|---|---|
| RCT | 现有治疗证据、不确定性、需要比较的干预 |
| 观察性研究 | 已有关联、因果机制、未解决混杂或人群缺口 |
| 诊断研究 | 现有测试性能、参考标准、目标使用场景 |
| 预后研究 | 已有预测因子或模型、验证不足、校准问题 |
| 系统综述 | 综述为何需要更新或重新评价 |
| 公共卫生研究 | 疾病负担、人群差异、政策或干预证据缺口 |
| 经济评价 | 现有成本效果证据、决策背景、资源约束 |
| 质性研究 | 已有量化证据不能解释的经验、意义或行为机制 |

### 8.3 好的证据缺口写法

弱写法：

```text
国内外研究较少。
```

强写法：

```text
既往研究多关注短期生物标志物变化，而较少评估患者重要结局，如住院、生活质量和严重不良事件。因此，现有证据不足以支持该干预在常规临床中的风险收益判断。
```

弱写法：

```text
该预测模型尚未被广泛研究。
```

强写法：

```text
虽然已有多个模型用于预测 30 天再入院风险，但多数仅报告区分度，缺少外部验证和校准评价，限制了其在不同医疗系统中的使用。
```

### 8.4 文献综述常见错误

- 罗列研究结果，不提研究设计质量；
- 只引用支持本文假设的研究；
- 不区分 RCT、观察性研究、系统综述和指南；
- 把低质量证据写成确定事实；
- 不说明已有研究人群与本文人群差异；
- 不说明结局差异；
- 不说明为何需要新研究；
- 把系统综述缺口和原始研究缺口混淆。

### Agent 质量判断

```text
1. 文献综述是否服务于研究问题？
2. 是否区分不同证据层级和研究设计？
3. 是否说明已有证据的具体不足？
4. 是否自然导出本文目的？
5. Discussion 是否回到这些文献比较？
```

## 9. 结果、分析或论证主体的组织方式

### 9.1 Results 不应按统计软件输出顺序组织

弱结构：

```text
3.1 一般资料
3.2 单因素分析
3.3 多因素分析
3.4 ROC 曲线
```

强结构：

```text
3.1 研究对象筛选与基线特征
3.2 干预对预设主要结局的影响
3.3 次要结局和安全性结果
3.4 预设亚组分析未显示明确效应修饰
3.5 敏感性分析支持主结果稳健性
```

### 9.2 临床试验 Results 组织

```text
1. Participant flow
2. Recruitment and follow-up
3. Baseline characteristics
4. Adherence / protocol deviations
5. Primary outcome
6. Secondary outcomes
7. Harms
8. Subgroup and sensitivity analyses
```

### 9.3 观察性研究 Results 组织

```text
1. Study population
2. Exposure distribution
3. Baseline differences
4. Outcome incidence
5. Main association
6. Adjusted analyses
7. Sensitivity analyses
8. Negative control or additional analyses, if applicable
```

### 9.4 诊断研究 Results 组织

```text
1. Participant flow
2. Index test and reference standard completion
3. 2×2 table
4. Sensitivity, specificity, likelihood ratios
5. ROC / AUC, if applicable
6. Indeterminate results
7. Subgroup or threshold analyses
```

### 9.5 预后 / 预测模型 Results 组织

```text
1. Participant characteristics
2. Outcome incidence
3. Predictor availability and missingness
4. Model specification
5. Apparent performance
6. Internal validation
7. External validation
8. Calibration
9. Clinical utility
```

### 9.6 系统综述 Results 组织

```text
1. Study selection
2. Study characteristics
3. Risk of bias
4. Results of individual studies
5. Synthesis results
6. Heterogeneity
7. Sensitivity / subgroup analysis
8. Publication bias
9. Certainty of evidence
```

### 9.7 医学图表叙事

常见主图表：

| 图表 | 功能 |
|---|---|
| 流程图 | 显示筛选、纳入、随访和分析人数 |
| Table 1 | 描述基线特征 |
| Table 2 | 主要和次要结局 |
| Figure 1 | 主要结果图、森林图或生存曲线 |
| Figure 2 | 亚组分析 |
| Supplement | 敏感性分析、额外模型、定义细节 |

表格和图不应只作为数据容器，而应服务于“可信证据链”。

### 9.8 常见错误

- 把所有统计结果等量呈现；
- 先报告探索性结果，后报告主要结局；
- Table 1 过长但缺少关键变量；
- 只报告 OR，不报告绝对风险；
- 亚组分析图很显眼，但主结果不稳；
- 敏感性分析放附录但正文不解释；
- 不良事件报告不足；
- 诊断研究没有 2×2 数据；
- 预测模型只报告 AUC，不报告校准。

## 10. 讨论与结论写法

### 10.1 Discussion 的四层提升

```text
结果层：主要发现是什么？
临床层：效应大小是否对患者或人群重要？
证据层：与已有研究和证据体系如何关系？
边界层：偏倚、混杂和外部效度如何影响解释？
```

### 10.2 临床意义表达公式

```text
虽然结果达到统计学显著性，但绝对效应较小，临床意义可能有限。
```

```text
结果未达到统计学显著性，但置信区间仍包含具有临床意义的获益和伤害，因此不能排除重要效果。
```

```text
该结果提示潜在获益，但由于样本量有限且随访时间较短，不能据此支持常规临床应用。
```

### 10.3 观察性研究结论表达

不宜写：

```text
X 导致 Y。
```

更稳妥：

```text
X 与 Y 风险升高相关。尽管本研究调整了多个潜在混杂因素，仍不能排除残余混杂和测量误差。
```

### 10.4 诊断研究结论表达

不宜写：

```text
该测试可替代现有诊断标准。
```

更稳妥：

```text
该测试在本文人群中显示较高敏感度和特异度，但是否可替代现有诊断流程仍需在实际临床路径中评估其对患者结局和资源使用的影响。
```

### 10.5 预测模型结论表达

不宜写：

```text
该模型可用于临床决策。
```

更稳妥：

```text
该模型在外部验证集中表现出可接受的区分度和校准度，但在临床使用前仍需前瞻性影响研究和可实施性评估。
```

### 10.6 结论边界

医学结论应限定：

- 人群；
- 场景；
- 时间；
- 干预强度；
- 数据来源；
- 结局；
- 随访长度；
- 医疗系统；
- 统计不确定性。

## 11. 审稿人关注点

### 11.1 研究问题是否重要

审稿人会问：

```text
1. 该问题是否影响临床或公共卫生决策？
2. 是否已有足够证据？
3. 本研究是否只是重复低质量研究？
4. 研究结局是否对患者或人群重要？
```

### 11.2 设计是否匹配问题

```text
1. 疗效问题是否用随机或准实验设计？
2. 因果问题是否只用横断面数据？
3. 诊断研究是否有合适参考标准？
4. 预测模型是否有验证？
5. 系统综述是否有完整检索和偏倚评价？
```

### 11.3 样本是否充分

```text
1. 样本量是否估算？
2. 事件数是否足够？
3. 失访是否严重？
4. 亚组分析是否样本过小？
5. 置信区间是否过宽？
```

### 11.4 偏倚和混杂是否处理

```text
1. 选择偏倚如何控制？
2. 暴露和结局测量是否可靠？
3. 关键混杂因素是否调整？
4. 是否存在残余混杂？
5. 是否有敏感性分析？
```

### 11.5 统计分析是否适当

```text
1. 模型是否匹配数据结构？
2. 是否考虑聚类、重复测量或时间到事件？
3. 是否处理缺失数据？
4. 是否报告效应量和 CI？
5. 是否过度依赖 P 值？
```

### 11.6 报告是否透明

```text
1. 是否匹配报告规范？
2. 是否有流程图？
3. 是否说明伦理审批？
4. 是否有注册号或方案？
5. 是否报告所有预设结局？
6. 是否披露资金和利益冲突？
```

### 11.7 结论是否过度

```text
1. 观察性研究是否写成因果？
2. 替代结局是否写成患者获益？
3. 诊断准确性是否写成改善结局？
4. 统计显著是否写成临床重要？
5. 小样本或单中心是否推广过大？
```

## 12. 常见失败模式

### 12.1 研究类型不清

表现：

- 文中没有明确是队列、病例对照、横断面还是试验；
- 报告规范无法匹配；
- Methods 混乱。

修正：

```text
1. 明确研究设计。
2. 根据设计重写 Methods。
3. 匹配报告规范。
4. 调整结论强度。
```

### 12.2 把观察性关联写成因果

修正：

```text
1. 改用“相关”“关联”“风险较高 / 较低”。
2. 说明混杂控制。
3. 讨论残余混杂。
4. 如果有因果推断框架，说明目标试验模拟、DAG 或识别假设。
```

### 12.3 只报告 P 值

修正：

```text
1. 补充效应量。
2. 补充 95% CI。
3. 补充绝对风险或均值差。
4. 讨论临床重要性。
```

### 12.4 主要结局不清

修正：

```text
1. 明确一个主要结局。
2. 标注次要和探索性结局。
3. 检查是否与方案或注册一致。
4. 避免事后选择显著结局。
```

### 12.5 混杂控制不足

修正：

```text
1. 先验列出混杂因素。
2. 说明协变量选择依据。
3. 避免调整中介或碰撞变量。
4. 增加敏感性分析。
```

### 12.6 缺失数据被忽略

修正：

```text
1. 报告缺失比例。
2. 说明缺失机制假设。
3. 使用合理处理方法。
4. 做敏感性分析。
```

### 12.7 不良事件报告不足

修正：

```text
1. 报告总不良事件和严重不良事件。
2. 说明事件判定方式。
3. 报告停药或退出原因。
4. 说明样本量是否足以评价安全性。
```

### 12.8 过度临床建议

修正：

```text
1. 将“应推广”改为“提示可能有价值”。
2. 说明证据等级和限制。
3. 提出需要进一步验证的研究。
```

## 13. Agent 写作辅助策略

## 13.1 Agent 总体流程

```text
1. 识别研究类型。
2. 匹配报告规范。
3. 提取 PICO / PECO / PICOTS。
4. 检查研究设计是否匹配问题。
5. 检查研究对象、纳排标准和流程。
6. 检查暴露 / 干预 / 测试 / 预测因素。
7. 检查结局定义。
8. 检查偏倚、混杂、失访和缺失数据。
9. 检查统计方法。
10. 检查 Results 是否完整透明。
11. 检查 Discussion 是否区分统计与临床意义。
12. 检查结论是否过度外推。
```

### 13.2 研究类型与报告规范匹配

Agent 应提醒用户根据研究类型选择规范，而不是泛泛要求“遵循医学论文规范”。

| 研究类型 | 可参考报告规范 / 工具 |
|---|---|
| 随机对照试验 | CONSORT；方案用 SPIRIT；干预细节用 TIDieR |
| 观察性研究 | STROBE |
| 常规收集健康数据观察性研究 | STROBE + RECORD |
| 诊断准确性研究 | STARD |
| 病例报告 | CARE |
| 系统综述 / Meta 分析 | PRISMA；证据确定性可用 GRADE |
| 健康经济学评价 | CHEERS |
| 质性访谈 / 焦点小组 | COREQ；广义质性研究可参考 SRQR |
| 预测模型 | TRIPOD+AI；偏倚评价可参考 PROBAST / PROBAST+AI |
| 医疗质量改进 | SQUIRE |

### 13.3 Agent 检查结论是否过度外推

高风险表达：

```text
证明有效
可广泛应用
建议临床推广
显著改善预后
独立危险因素
因果关系
替代金标准
适用于所有患者
安全可靠
```

改写规则：

| 过强表达 | 更稳妥表达 |
|---|---|
| X 导致 Y | X 与 Y 相关 |
| 证明该干预有效 | 结果支持该干预在本文人群中可能有效 |
| 可广泛推广 | 需要在更多场景和人群中验证 |
| 安全 | 未观察到明显安全性信号，但样本量不足以排除罕见风险 |
| 替代金标准 | 可能作为辅助测试，仍需临床路径评价 |
| 独立危险因素 | 在调整所纳入协变量后仍与结局相关 |

### 13.4 Agent 生成“结论支持矩阵”

| 论文结论 | 支持结果 | 研究设计能否支持 | 风险 | 建议 |
|---|---|---|---|---|
| 干预有效 | RCT 主要结局显著 | 较强 | 样本量 / 随访 | 补临床意义 |
| 暴露导致结局 | 回顾性关联 | 弱 | 残余混杂 | 改为关联 |
| 诊断测试可替代标准 | 敏感度 / 特异度高 | 不足 | 未评价临床结局 | 改为辅助 |
| 模型可用于临床 | 外部验证有限 | 不足 | 无影响研究 | 限定为需进一步验证 |
