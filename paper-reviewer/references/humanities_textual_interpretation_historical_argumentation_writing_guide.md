# 人文学科 / 文本解释 / 历史论证型论文写作指南

## 1. 适用范围

本指南适用于以**文本、史料、作品、概念、思想传统、艺术对象、文化现象、语言材料或历史语境**为核心研究对象，并通过解释、论证、细读、语境化和概念辨析生产知识的论文。

适用领域包括但不限于：

- 文学研究；
- 历史学；
- 哲学；
- 艺术史；
- 宗教学；
- 古典学；
- 思想史；
- 文化研究；
- 语言学中的文本、语义、语用、话语和历史语言方向；
- 戏剧、电影、媒介和视觉文化研究；
- 音乐学和表演研究；
- 翻译研究中的文本解释与文化阐释方向；
- 区域研究中的历史、文本和思想传统研究；
- 其他以解释性论证为核心的人文学科论文。

本指南适用于以下论文类型：

- 文本细读论文；
- 作品阐释论文；
- 历史论证论文；
- 思想史论文；
- 概念史论文；
- 哲学论证论文；
- 艺术作品分析论文；
- 档案史料解释论文；
- 文化现象阐释论文；
- 话语和修辞分析论文；
- 理论介入型人文论文；
- 比较文本或比较思想论文；
- 对既有解释提出修正的论文；
- 重新解释经典文本、史料或作品的论文。

这类论文的核心不是“展示材料很多”，也不是“套用某个理论”，而是：

> 以材料解释推进一个有争议、有新意、可论证的中心论点。

Agent 辅助这类论文时，应重点判断：

```text
研究对象是否明确？
问题是否具有解释价值？
作者是否进入了已有学术争论？
中心论点是否清晰、有新意、可辩护？
材料是否被解释而不是堆砌？
理论是否服务于材料，而不是压倒材料？
每一节是否推进主论点？
是否回应了可能的替代解释？
结论是否真正改变了读者对对象的理解？
```

## 2. 不适用范围

本指南不应机械套用于以下文本。

### 2.1 自然科学实验论文

自然科学实验论文通常通过实验设计、测量、数据和结果讨论生产知识。人文学科论文一般不以实验可重复性为核心，也不应强行套用 Introduction-Methods-Results-Discussion 的 IMRaD 结构。

### 2.2 定量社会科学论文

如果论文主要通过变量、样本、模型和统计识别检验假设，应使用定量社会科学论文指南。  
人文学科论文可以讨论社会现象，但其证据通常不是统计模型，而是文本、史料、概念和解释性论证。

### 2.3 纯资料汇编或史料整理

如果文本只是整理材料、编年、目录、译注或文献汇编，而没有中心论点和解释性推进，则不构成完整人文学术论文。

资料整理可以是重要学术工作，但作为论文仍需说明：

```text
这些材料改变了我们对什么问题的理解？
它们支持什么新解释？
它们挑战了哪些既有判断？
```

### 2.4 读书报告或作品赏析

读书报告通常总结作品内容和个人感受；人文学术论文必须提出可论证的解释问题，并与已有研究对话。

弱写法：

```text
本文分析《红楼梦》中的女性形象。
```

较强写法：

```text
本文论证《红楼梦》中的女性书写并非单纯体现才情理想，而是通过诗社、病体与家族秩序之间的张力，揭示了清代闺阁文化中才名与规训的双重结构。
```

### 2.5 纯理论宣言

如果文本只是用宏大理论表达立场，而没有具体材料解释、文本分析或概念论证，则不适合本指南所指的人文论文范式。

### 2.6 文艺评论或随笔

评论和随笔可以富有洞见，但学术论文需要更明确的研究对象、学术争论、论证结构、证据使用和引用规范。

### 2.7 政策评论或公共议论文

公共评论强调立场和说服公众；人文学术论文强调材料解释、学术定位和论证可检验性。

## 3. 知识生产逻辑

人文学科 / 文本解释 / 历史论证型论文通过以下方式生产可信知识：

```text
研究对象
→ 解释问题
→ 学术争论
→ 中心论点
→ 材料选择
→ 文本细读 / 史料解释 / 概念分析 / 语境化
→ 子论点推进
→ 替代解释回应
→ 理论或历史意义
→ 结论
```

### 3.1 这类论文依靠什么产生可信知识

| 可信性来源 | 写作中应体现什么 | Agent 判断重点 |
|---|---|---|
| 明确对象 | 文本、史料、作品、概念、事件、传统或现象 | 对象是否具体 |
| 解释问题 | 为什么这个对象需要重新理解 | 是否有问题意识 |
| 学术争论 | 已有研究如何解释，分歧在哪里 | 是否进入争论 |
| 中心论点 | 作者对对象提出什么可辩护解释 | 是否有 thesis |
| 材料解释 | 细读、史料分析、语境化、概念辨析 | 是否解释而非堆砌 |
| 论证推进 | 每节有子论点，逐步支持主论点 | 是否有论证链 |
| 替代解释 | 回应可能反驳或不同读法 | 是否处理异议 |
| 理论敏感性 | 理论帮助看见材料中的问题 | 是否理论套话 |
| 历史语境 | 材料放在时代、制度、语言、思想背景中理解 | 是否脱离语境 |
| 解释边界 | 说明论点适用范围和限制 | 是否过度概括 |

### 3.2 人文学科论文为什么不能套用 IMRaD

IMRaD 结构适合许多实验、临床和经验研究论文，因为它围绕：

```text
研究问题
→ 方法
→ 结果
→ 讨论
```

但人文学科论文通常不是通过“方法产生结果”来组织，而是通过**论证推进**来组织。它的正文不是“结果展示”，而是逐步展开解释。

人文学科论文的常见逻辑是：

```text
提出问题
→ 定位争论
→ 表明中心论点
→ 解释材料
→ 逐节推进子论点
→ 处理反驳
→ 重申理论或历史意义
```

因此，它更接近 thesis-driven argument，而不是 IMRaD。

### 3.3 人文学科论文的核心闭环

强论文应形成以下闭环：

```text
对象为什么值得重新解释？
已有研究如何理解？
本文不同意、补充或重构哪里？
中心论点是什么？
哪些材料支持它？
每一节如何推进它？
可能的反读如何回应？
结论如何改变学术理解？
```

常见断裂包括：

```text
材料很多，但没有中心论点。
理论很大，但没有细读。
文献综述很长，但没有进入争论。
每节都像独立小论文，缺少论证推进。
结论只是总结内容，没有说明解释贡献。
```

### 3.4 人文学科论文中的 Claim-Evidence-Warrant

人文学科论文也需要“主张—证据—解释桥梁”。

| 元素 | 人文学科论文中的对应 |
|---|---|
| Claim | 对文本、史料、概念或现象的解释性主张 |
| Evidence | 引文、史料、档案、作品细节、语词、图像、概念用法、历史事实 |
| Warrant | 作者说明该材料为何支持该解释 |
| Counter-reading | 可能的不同解释或反证 |
| Boundary | 论点适用的文本、时期、语境或传统范围 |

示例：

```text
Claim：《某文本》中的“沉默”并非人物被动性的表现，而是一种对礼法秩序的间接抵抗。
Evidence：文本多次将人物沉默放置在权威话语之后，并通过叙述视角强调沉默造成的关系断裂。
Warrant：如果沉默只是顺从，叙述不必持续标记其对场景秩序的扰动；这些叙事安排使沉默成为一种低可见度的行动。
Counter-reading：沉默也可被读作女性无权发声的结果。
Response：本文并不否认其被压制的一面，而是指出文本同时赋予沉默以扰动叙事秩序的功能。
```

Agent 检查规则：

```text
每当作者提出解释性判断时，检查：
1. 具体材料在哪里？
2. 该材料是否被细读？
3. 作者是否解释材料如何支持判断？
4. 是否存在不同读法？
5. 作者是否回应了不同读法？
6. 该判断是否推进中心论点？
```

## 4. 典型论文结构

人文学科论文结构通常围绕论点推进，而不是 IMRaD。常见结构如下：

```text
Title
Abstract
Keywords

1. Introduction
2. Scholarly Debate / Literature Review
3. Thesis Statement / Argument Overview
4. Context / Materials
5. Argument Section I
6. Argument Section II
7. Argument Section III
8. Counter-reading / Alternative Interpretation
9. Discussion / Implications
10. Conclusion

References / Works Cited
Appendix, if needed
```

不同学科会有变体：

- 文学论文常将文本细读分布在正文各节；
- 历史论文常按时间、问题、材料类型或论证阶段组织；
- 哲学论文常按问题、概念、论证、反驳、回应组织；
- 艺术史论文常围绕作品、图像、媒介、观看制度和历史语境展开；
- 思想史论文常围绕概念迁移、语义变化、思想语境和文本关系展开；
- 宗教学论文常结合文本传统、仪式实践、教义解释和历史语境；
- 古典学论文常强调文本训诂、版本、语源、注疏传统和历史背景。

### 4.1 章节功能总览

| 章节 | 核心功能 | 不应写成 |
|---|---|---|
| Introduction | 提出对象、问题、争议、中心论点 | 泛泛背景介绍 |
| Scholarly Debate | 进入学术争论，说明本文位置 | 研究史罗列 |
| Thesis Statement | 明确本文要证明什么 | 模糊主题说明 |
| Context / Materials | 交代材料、语境、文本版本、史料来源 | 无关背景堆砌 |
| Argument Sections | 每节推进一个子论点 | 材料汇编 |
| Counter-reading | 回应可能反读或替代解释 | 忽略异议 |
| Discussion | 提炼理论、历史或解释意义 | 重复正文 |
| Conclusion | 回到问题，压缩贡献和边界 | 空泛总结 |

### 4.2 典型论证结构

#### 结构一：文本细读型

```text
Introduction：提出文本解释问题
Debate：既有读法与争议
Thesis：本文新读法
Section 1：关键词 / 意象 / 叙事结构
Section 2：人物关系 / 修辞策略
Section 3：文本与历史语境
Counter-reading：回应既有读法
Conclusion：说明新解释的意义
```

#### 结构二：历史论证型

```text
Introduction：提出历史问题
Debate：史学争论
Sources：史料类型和使用方式
Section 1：既有解释的问题
Section 2：新史料或新语境
Section 3：重新解释事件、制度或行动者
Section 4：更广泛历史意义
Conclusion
```

#### 结构三：哲学 / 概念分析型

```text
Introduction：提出概念或论证难题
Debate：已有解释或争论
Thesis：本文立场
Section 1：概念区分
Section 2：重构论证
Section 3：回应反驳
Section 4：推论或应用
Conclusion
```

#### 结构四：艺术史 / 视觉文化型

```text
Introduction：提出作品或视觉问题
Historiography：既有解释
Materials：作品、图像、媒介和语境
Section 1：形式分析
Section 2：观看制度 / patronage / 技术条件
Section 3：图像意义与历史语境
Counter-reading
Conclusion
```

## 5. 各章节写作指南

## 5.1 标题 Title

### 写作目标

标题应准确传达研究对象、解释问题、核心论点或理论视角。

### 常见标题类型

| 类型 | 结构 | 示例形式 |
|---|---|---|
| 对象 + 问题 | X 中的 Y 问题 | 某文本中的时间经验 |
| 概念 + 对象 | “X”的生成：Y 中的 Z | 核心概念提炼 |
| 争议介入型 | 重新理解 X | 对既有读法修正 |
| 主副标题 | 概念性主标题 + 解释性副标题 | “沉默的行动”：某作品中的女性发声 |
| 历史语境型 | X 与 Y 时期的 Z | 思想或制度语境 |
| 比较型 | X 与 Y 之间 | 跨文本、跨传统比较 |

### 常见错误

- 标题只有对象，没有问题；
- 标题过于文学化，读者看不出研究对象；
- 标题像读书报告，如“论某作品的艺术特色”；
- 标题声称宏大理论，但正文材料很窄；
- 标题不体现核心论点；
- 标题中使用空泛词，如“浅析”“探讨”“研究”。

### Agent 质量判断

```text
1. 标题是否说明研究对象？
2. 是否体现解释问题或论点？
3. 是否避免空泛和过度文学化？
4. 是否与正文论证范围一致？
5. 是否能进入学术检索和争论？
```

## 5.2 摘要 Abstract

### 写作目标

摘要应压缩论文的解释性论证链：

```text
对象
→ 学术争议 / 解释缺口
→ 中心论点
→ 材料和方法
→ 论证路径
→ 贡献
```

### 推荐结构

```text
1. 对象句：说明本文研究什么文本、史料、作品、概念或现象。
2. 争议句：说明已有研究如何解释，问题在哪里。
3. 论点句：明确本文中心论点。
4. 材料句：说明本文使用哪些材料或分析对象。
5. 论证句：概括正文如何推进论点。
6. 贡献句：说明本文改变了什么理解。
```

### 示例骨架

```text
本文重新考察 X 文本中的 Y 意象。既有研究多将 Y 理解为 A 的象征，但较少注意其在叙事结构中与 B 之间的关系。本文论证，Y 并非单纯承担 A 功能，而是通过反复中断叙事时间，构成对 C 秩序的隐性批评。文章通过对三个关键段落的细读，并结合 D 时期的礼法语境，说明 Y 如何在文本内部形成双重意义。本文的贡献在于将关于 X 的研究从主题象征读法推进到叙事时间和社会秩序关系的讨论。
```

### 常见错误

- 摘要只是介绍对象；
- 不说明学术争议；
- 不给中心论点；
- 只说“本文分析……”；
- 贡献写成“具有重要意义”；
- 不说明材料或论证路径；
- 摘要使用过多理论术语但无具体对象。

### Agent 质量判断

```text
1. 摘要是否明确对象？
2. 是否说明已有解释缺口？
3. 是否有清晰中心论点？
4. 是否说明材料和论证方式？
5. 是否说明贡献？
6. 是否避免空泛介绍？
```

## 5.3 Introduction 引言

### 写作目标

Introduction 应提出对象、问题、争议和中心论点。它的功能不是铺陈时代背景，而是让读者知道：

```text
本文研究什么？
为什么这个对象需要重新解释？
已有研究如何理解它？
本文不同意、补充或推进哪里？
本文的核心论点是什么？
正文将如何证明？
```

### 推荐组织方式

```text
1. 引入研究对象；
2. 指出一个解释性问题或矛盾；
3. 简述已有解释或学术争议；
4. 提出本文中心论点；
5. 说明材料和方法；
6. 预告论证结构；
7. 声明贡献。
```

### 好的引言开头

弱写法：

```text
《某作品》是中国现代文学史上的重要作品，自发表以来受到广泛关注。
```

强写法：

```text
《某作品》中最反复出现的并不是通常被研究者强调的革命叙事，而是人物对日常物件的凝视。这个看似边缘的细节，恰恰暴露了文本在政治动员与私人经验之间的张力。
```

弱写法：

```text
历史上关于某事件的研究很多，但仍有不足。
```

强写法：

```text
关于某事件，既有研究多从国家制度转型解释其发生原因。然而，新近公开的地方档案显示，地方官员对制度命令的解释并不一致，这使单一的中央政策解释难以充分说明事件的实际展开过程。
```

### 中心论点 Thesis Statement

人文学科论文必须有中心论点，而不是只有主题。

主题：

```text
本文研究《某作品》中的空间书写。
```

论点：

```text
本文论证，《某作品》中的空间书写并非背景描写，而是一种叙事装置：它通过不断压缩人物行动范围，表现现代城市经验中的道德困境。
```

好的中心论点应满足：

```text
1. 可争论；
2. 可被材料支持；
3. 有新意；
4. 不是事实陈述；
5. 能统领全文；
6. 明确指出本文要改变什么理解。
```

### 常见错误

- 引言过长，中心论点出现太晚；
- 只有研究对象，没有问题；
- 文献综述挤占引言；
- 论点模糊，如“本文试图探讨……”；
- 论点不可争议，只是事实；
- 引言没有说明学术争议；
- 引言提出的问题正文没有回应；
- 用理论术语替代论点。

### Agent 质量判断

```text
1. 引言是否明确研究对象？
2. 是否提出解释性问题？
3. 是否进入学术争议？
4. 是否有清晰 thesis statement？
5. 是否说明材料和方法？
6. 是否预告论证路径？
7. 是否避免泛泛背景？
```

---

## 5.4 Scholarly Debate / Literature 文献综述与学术争论

### 写作目标

人文学科文献综述的目标不是罗列研究史，而是进入学术争论。它应说明：

```text
已有研究如何解释对象？
这些解释之间有什么分歧？
哪些解释有盲点？
本文如何介入？
```

### 文献综述应包含

- 关键研究传统；
- 主要解释路径；
- 争论焦点；
- 与本文对象最相关的研究；
- 本文与已有解释的关系；
- 本文的切入点。

### 组织方式

#### 方式一：解释路径式

```text
象征主义读法
历史语境读法
叙事结构读法
本文的综合或修正
```

#### 方式二：争议式

```text
观点 A：文本支持某种意识形态
观点 B：文本包含反讽或抵抗
本文：文本在两者之间形成张力
```

#### 方式三：方法式

```text
文本内在细读
历史语境研究
理论化解释
本文结合细读与语境化
```

#### 方式四：问题式

```text
已有研究解释了人物形象
已有研究解释了历史背景
但尚未解释叙事形式如何组织历史经验
```

### 好的文献综述段落

弱写法：

```text
A 研究了 X，B 研究了 Y，C 认为 Z。
```

强写法：

```text
关于该文本中的城市空间，研究大致形成两种读法。一类研究将城市视为现代性经验的象征，强调其压迫性和异化功能；另一类研究则关注城市空间中的消费、流动和感官经验，认为文本展现了现代主体的生成。两种读法都揭示了空间的重要性，但它们较少解释空间如何在叙事层面调节人物伦理选择。本文正是从这一点切入。
```

### 常见错误

- 按研究者逐个介绍；
- 文献综述像研究史年表；
- 只说前人不足，不说明争论；
- 把所有文献都写成“没有研究本文问题”；
- 不处理最重要的相反观点；
- 文献综述与中心论点脱节；
- 不说明本文介入位置；
- 对已有研究过度贬低。

### Agent 质量判断

```text
1. 文献综述是否围绕争论组织？
2. 是否概括已有解释路径？
3. 是否说明本文介入点？
4. 是否处理最接近和相反观点？
5. 是否避免罗列研究史？
6. 是否为 thesis statement 服务？
```

## 5.5 Thesis Statement 中心论点

### 写作目标

中心论点是全文的发动机。人文学科论文中的 thesis 不只是“研究目的”，而是作者要说服读者接受的解释性判断。

### 好 thesis 的标准

```text
1. 对特定对象提出解释；
2. 可以被反驳或争论；
3. 超出显而易见事实；
4. 能由材料支持；
5. 有学术新意；
6. 能统领全文结构；
7. 能回答“so what?”。
```

### 弱 thesis 与强 thesis

弱：

```text
本文分析某作品中的女性形象。
```

强：

```text
本文论证，某作品中的女性形象并非民族寓言的被动承载者，而是通过沉默、拒绝和重复劳动等低可见度行动，暴露了民族叙事内部对性别劳动的依赖。
```

弱：

```text
本文研究某思想家的自由观。
```

强：

```text
本文论证，某思想家的自由观并非从个人权利出发，而是在礼法秩序的重构中定义行动者的自我限制；这一点使其不同于近代自由主义的主体假设。
```

弱：

```text
本文探讨某事件的原因。
```

强：

```text
本文论证，某事件并非中央政策直接导致，而是在地方财政压力、基层官员晋升逻辑和宗族调解机制失效三者交汇处形成。
```

### Thesis 与主题、问题、结论的区别

| 类型 | 示例 | 功能 |
|---|---|---|
| 主题 | 某作品中的空间书写 | 研究范围 |
| 问题 | 空间书写如何组织人物经验？ | 研究疑问 |
| Thesis | 空间书写通过压缩行动范围表现伦理困境 | 中心论点 |
| 结论 | 该读法改变了我们对作品现代性经验的理解 | 贡献收束 |

### Agent 判断规则

```text
如果一句话不能被合理学者反驳，它通常不是 thesis。
如果一句话只是“本文研究……”，它不是 thesis。
如果一句话不能统领正文结构，它不是有效 thesis。
```

## 5.6 Context / Materials 语境与材料

### 写作目标

Context / Materials 部分说明论文的证据基础和解释环境。它应回答：

```text
本文使用哪些文本、史料、作品或概念材料？
为什么选择这些材料？
材料的版本、来源、时间和语境是什么？
读者需要哪些背景才能理解论证？
```

### 文本材料应说明

- 作品版本；
- 发表时间；
- 文本来源；
- 翻译版本，如适用；
- 版本差异；
- 作者、出版、传播背景；
- 文类和体裁；
- 与其他文本的关系。

### 历史史料应说明

- 档案来源；
- 史料类型；
- 形成时间；
- 作者或机构；
- 保存和传播条件；
- 史料偏向；
- 史料空白；
- 史料与问题的关系。

### 艺术作品应说明

- 作品名称；
- 年代；
- 材质；
- 尺寸；
- 委托关系；
- 展示或观看语境；
- 图像传统；
- 修复或收藏历史。

### 哲学和思想史材料应说明

- 文本版本；
- 概念使用语境；
- 论争对象；
- 思想传统；
- 术语翻译；
- 同时期相关文本。

### 语境写作原则

语境不是背景堆砌，而应服务论点。

弱写法：

```text
某时期政治、经济、文化都发生了巨大变化。
```

强写法：

```text
某时期地方财政制度的变化直接改变了县级官员处理民间纠纷的激励结构。这一制度背景有助于解释史料中官员为何反复强调“息讼”而非追求正式审判。
```

### 常见错误

- 不说明文本版本；
- 不说明史料来源；
- 语境过多但与论点无关；
- 将史料视为透明事实；
- 忽略材料形成条件；
- 忽略翻译和版本问题；
- 只引用二手材料；
- 不解释材料选择理由。

### Agent 质量判断

```text
1. 材料是否明确？
2. 材料来源和版本是否说明？
3. 语境是否服务论点？
4. 是否说明材料局限和偏向？
5. 是否避免背景堆砌？
```

## 5.7 Argument Sections 论证主体

### 写作目标

正文小节不是材料分类，而是论证推进。每一节都应推进一个子论点，并共同支持中心论点。

每节应回答：

```text
本节要证明什么？
它如何支持中心论点？
使用哪些材料？
材料如何被解释？
本节如何过渡到下一节？
```

### 子论点结构

推荐每节采用：

```text
1. 小节标题表达子论点；
2. 段首提出本节主张；
3. 呈现材料；
4. 解释材料；
5. 处理可能反读；
6. 小结本节如何推进中心论点。
```

### 弱小节标题

```text
一、文本分析
二、人物形象
三、艺术特色
```

### 强小节标题

```text
一、被压缩的行动空间：城市叙事中的伦理限制
二、沉默作为行动：女性主体的低可见度抵抗
三、从物件到制度：日常细节中的权力结构
```

### 段落论证结构

人文学科论文段落可采用：

```text
子主张
→ 材料引用或描述
→ 细读 / 解释
→ 与中心论点连接
→ 过渡
```

示例：

```text
文本在这里并不只是描写夜色，而是通过视觉不可辨认性制造道德判断的悬置。叙述者反复使用“模糊”“看不清”“仿佛”等词，使人物行动脱离明确因果判断。正是在这种叙述模糊中，作品将道德困境从人物性格问题转化为现代城市经验的问题。
```

### 每节推进方式

常见推进逻辑：

| 推进方式 | 适用 |
|---|---|
| 从词语到结构 | 文本细读 |
| 从个案到制度 | 历史论文 |
| 从概念区分到论证重构 | 哲学论文 |
| 从形式到语境 | 艺术史 |
| 从局部细节到整体解释 | 文学研究 |
| 从旧读法到新读法 | 争议介入 |
| 从材料矛盾到重新解释 | 史料解释 |
| 从反例到理论修正 | 思想史或哲学 |

### 常见错误

- 小节按材料类型排列，而不是按论点推进；
- 每节像独立短文；
- 段落没有主题句；
- 引文后没有解释；
- 细读停留在复述；
- 每段都引入新观点但不回到主论点；
- 过渡缺失；
- 结尾没有小结。

### Agent 质量判断

```text
1. 每节是否有明确子论点？
2. 子论点是否支持中心论点？
3. 材料是否被解释？
4. 每段是否有 claim-evidence-warrant？
5. 小节之间是否有递进关系？
6. 是否避免材料分类式结构？
```

## 5.8 文本细读、史料解释、概念分析与语境化

### 5.8.1 文本细读 Close Reading

文本细读不是摘录文本，而是分析文本如何通过语言、形式、叙事、修辞、声音、节奏、意象、体裁和结构产生意义。

细读可关注：

- 关键词；
- 反复出现的意象；
- 叙述视角；
- 语气；
- 修辞；
- 句法；
- 节奏；
- 隐喻；
- 对话结构；
- 沉默和省略；
- 叙事顺序；
- 文体转换；
- 互文关系；
- 体裁惯例；
- 译词和语义漂移。

弱细读：

```text
这里出现了“黑夜”，说明气氛压抑。
```

强细读：

```text
“黑夜”在这一段并非单纯营造压抑气氛。它与“看不清”“迟疑”“误认”等词连续出现，使视觉障碍成为叙事判断的条件。文本因此将人物的道德判断转化为感知不确定性问题。
```

### 5.8.2 史料解释

史料不是事实本身，而是历史情境中形成的材料。解释史料时应问：

```text
谁写的？
为谁写的？
在什么制度和语境中写的？
它说了什么？
它没有说什么？
它为何这样说？
它与其他材料如何印证或冲突？
```

弱史料使用：

```text
档案显示地方官员反对该政策。
```

强史料使用：

```text
档案中地方官员使用“民情未协”“恐滋扰累”等表述，并未直接反对政策本身，而是将反对理由转化为地方秩序风险。这表明地方执行者是在合法性语言内部重新定义政策可行性。
```

### 5.8.3 概念分析

概念分析关注：

- 概念定义；
- 语义范围；
- 历史变迁；
- 与相近概念的区别；
- 概念在不同文本中的用法；
- 概念背后的规范假设；
- 翻译和跨语境转换；
- 概念如何组织论证。

示例：

```text
本文区分“自由”作为权利主张和“自由”作为道德修养状态的两种用法。前者强调外在限制的移除，后者则强调主体对欲望的自我约束。该区分有助于解释某思想家为何同时批评专制权力和无节制个体主义。
```

### 5.8.4 历史语境化

语境化不是简单说明时代背景，而是说明材料在特定历史条件下如何产生意义。

应避免：

```text
由于时代背景如此，所以文本必然如此。
```

应写成：

```text
在某种制度、话语、出版、阅读、宗教、经济或政治条件下，文本中的某种表达获得了特定意义。
```

### 5.8.5 理论视角

理论不是现成标签。理论应帮助提出问题、组织材料和解释意义。

弱理论使用：

```text
根据福柯的权力理论，文本体现了规训。
```

强理论使用：

```text
本文借用福柯关于规训权力的视角，但不将文本简单归入规训范式。相反，本文关注文本如何通过自我记录、身体姿态和空间布置，使人物主动参与自身可见性的生产。
```

### Agent 质量判断

```text
1. 文本引用是否被细读？
2. 史料是否被语境化？
3. 概念是否被辨析？
4. 理论是否服务材料？
5. 材料解释是否推进论点？
6. 是否避免把材料当作透明事实？
```

## 5.9 Counter-reading / Alternative Interpretation 替代解释与反读

### 写作目标

强人文学科论文应主动回应可能的不同读法。因为人文解释通常不是唯一的，论文可信度来自对竞争解释的处理。

### 可能的替代解释包括

- 既有学者的相反读法；
- 文本中不支持作者论点的片段；
- 史料中的矛盾证据；
- 概念的其他含义；
- 不同理论视角；
- 材料来源偏差；
- 作者过度解释的风险；
- 历史语境中的其他可能性。

### 回应替代解释的结构

```text
1. 公平呈现替代解释；
2. 承认其解释力；
3. 指出其无法解释的材料或问题；
4. 展示本文解释如何更好解释材料；
5. 必要时修正自己的论点。
```

### 示例

```text
一种可能的读法是将人物的沉默理解为完全的被动顺从。这一读法能够解释文本中人物缺少公开反抗语言的事实。然而，它无法解释叙述者为何反复将沉默放置在权威话语中断的时刻。本文的读法并不否认沉默的受压制性，而是进一步指出，文本同时让沉默承担了扰乱场景秩序的叙事功能。
```

### 常见错误

- 不回应反对观点；
- 把反方观点写得很弱；
- 只说“本文不同意”；
- 忽略明显反证；
- 回应替代解释后不修正主论点；
- 把不同读法视为错误而非解释竞争。

### Agent 质量判断

```text
1. 是否识别了最强替代解释？
2. 是否公平呈现？
3. 是否用材料回应？
4. 是否说明本文读法的优势？
5. 是否必要时限定或修正论点？
```

## 5.10 Discussion / Implications 讨论与意义

### 写作目标

Discussion 应说明本文论证对学术争论、理论概念、历史理解或文本阐释有什么意义。

它应回答：

```text
如果本文论点成立，我们需要如何重新理解对象？
它改变了哪一类既有解释？
它对某个理论、概念或历史叙事有什么影响？
它的边界在哪里？
```

### 理论意义写法

弱写法：

```text
本文丰富了相关研究。
```

强写法：

```text
本文的读法将该文本从“民族寓言”的解释框架中部分移出，转而强调其对日常伦理困境的叙事组织。这意味着文本中的政治性并不只存在于宏大象征层面，也存在于人物对日常行动可能性的感知之中。
```

### 历史意义写法

```text
本文对地方档案的重新解释表明，该事件并非中央政策在地方的简单贯彻或失败，而是地方官员在多重制度压力下重新定义政策可行性的结果。这一发现促使我们将制度史叙事从命令—执行模型转向解释—协商模型。
```

### 概念意义写法

```text
通过区分“自律”与“自由”的两种语义层次，本文说明某思想传统中的自由概念并不能直接等同于现代权利话语。该区分有助于避免将近代政治概念倒置投射到早期思想文本之中。
```

### 常见错误

- Discussion 重复正文；
- 理论意义过大；
- 没有回到学术争论；
- 只写“对后续研究有启示”；
- 不说明边界；
- 把个案解释写成普遍理论。

### Agent 质量判断

```text
1. Discussion 是否说明论点的学术意义？
2. 是否回到文献争论？
3. 是否说明理论或历史理解如何改变？
4. 是否限定适用范围？
5. 是否避免空泛贡献语？
```

## 5.11 Conclusion 结论

### 写作目标

Conclusion 应回到对象、问题和论点，压缩全文论证，并说明贡献和边界。

### 推荐结构

```text
1. 回到研究问题；
2. 重申中心论点；
3. 概括论证路径；
4. 说明贡献；
5. 指出边界；
6. 提出进一步研究方向。
```

### 示例

```text
本文重新解释了某文本中的沉默书写。与将沉默视为被动顺从的既有读法不同，本文论证沉默在文本中承担了中断权威话语、重组人物关系和暴露礼法秩序裂缝的叙事功能。通过对三个关键场景的细读，并结合晚清闺阁书写的历史语境，本文表明女性主体的行动并不总是以公开发声的形式出现。该解释有助于将相关研究从“发声 / 失声”的二分推进到对低可见度行动形式的分析。当然，本文主要关注单一文本内部的叙事机制，未来研究可比较同一时期其他文本中的沉默书写。
```

### 常见错误

- 只重复引言；
- 不重申中心论点；
- 不说明贡献；
- 引入新材料；
- 结论比正文更宏大；
- 用口号或感想结尾；
- 不说明边界。

### Agent 质量判断

```text
1. 结论是否回答了引言问题？
2. 是否压缩中心论点？
3. 是否概括论证路径？
4. 是否说明贡献？
5. 是否明确边界？
6. 是否避免新材料和过度扩展？
```

## 6. 常见贡献类型

人文学科 / 文本解释 / 历史论证型论文的贡献通常不是“发现了一个材料”，而是材料解释带来的理解改变。

### 6.1 新读法贡献

对经典文本、作品或史料提出新的解释。

示例：

```text
本文将某作品中的旅行叙事重新解释为身份失序的叙事机制，而非单纯的空间移动。
```

质量要求：

- 明确既有读法；
- 说明新读法与旧读法区别；
- 有细读支撑；
- 能解释旧读法解释不了的材料。

### 6.2 语境化贡献

将对象放入新的历史、制度、语言、宗教、思想或媒介语境中重新理解。

示例：

```text
本文将某画作置于城市观看制度变化中，说明其构图选择与公共展览空间的出现相关。
```

质量要求：

- 语境与材料直接相关；
- 不是背景堆砌；
- 说明语境如何改变解释。

### 6.3 概念辨析贡献

重新界定或区分核心概念。

示例：

```text
本文区分某思想家文本中“礼”作为制度秩序和作为身体实践的两种用法。
```

质量要求：

- 概念用法有材料依据；
- 与已有概念区分；
- 说明辨析带来的解释收益。

### 6.4 史料贡献

使用新史料或重新解释旧史料，改变对历史问题的理解。

示例：

```text
本文通过地方档案显示某政策执行并非线性下达，而是经过地方官员的风险翻译。
```

质量要求：

- 史料来源清楚；
- 史料解释细致；
- 不只是“发现了材料”；
- 说明材料改变了什么历史叙事。

### 6.5 争论介入贡献

介入已有学术争论，支持、修正或超越某种解释。

示例：

```text
本文认为关于某文本“保守 / 激进”的二分争论忽略了文本在叙事形式上对政治行动可能性的悬置。
```

质量要求：

- 准确呈现争论；
- 公平处理对立观点；
- 明确本文如何推进争论。

### 6.6 理论修正贡献

通过材料分析修正某理论在特定对象上的使用方式。

示例：

```text
本文说明将某文本简单纳入“规训”框架会遮蔽人物对可见性的主动管理，因此需要将规训理论与表演性实践结合。
```

质量要求：

- 理论使用具体；
- 材料能支撑修正；
- 不是简单贴标签。

### 6.7 比较贡献

通过比较文本、传统、时期、区域或媒介揭示差异和转化。

质量要求：

- 比较维度明确；
- 不是并列介绍；
- 比较产生新的解释问题。

### 6.8 方法贡献

提出新的材料处理、阅读方式或解释路径。

示例：

```text
本文将版本差异、读者批注和出版语境结合起来，重新解释某文本的意义生成过程。
```

质量要求：

- 方法服务论点；
- 说明新方法看到旧方法看不到的问题。

## 7. 证据与论证标准

## 7.1 强人文学科证据的特征

```text
1. 材料具体；
2. 引用准确；
3. 文本或史料被解释而非摆放；
4. 语境与材料相关；
5. 概念使用清楚；
6. 推理链条可追踪；
7. 处理替代解释；
8. 论点与材料范围匹配；
9. 每段服务中心论点；
10. 与已有研究形成对话。
```

### 强证据组合示例

| 主张 | 强证据组合 |
|---|---|
| 某意象具有叙事功能 | 多处文本细读 + 叙事结构分析 |
| 某概念有双重含义 | 术语用法比较 + 历史语境 |
| 某事件应重新解释 | 多种史料互证 + 对旧解释的反驳 |
| 某作品与观看制度有关 | 形式分析 + 展览史 + 赞助关系 |
| 某思想不同于既有归类 | 概念辨析 + 同时代论争文本 |

## 7.2 弱证据的特征

弱证据常表现为：

- 只引用材料，不解释；
- 只讲背景，不回到文本；
- 用理论术语替代分析；
- 用孤立例子支撑大结论；
- 不说明材料来源；
- 不处理版本和翻译；
- 对史料采取透明事实观；
- 忽视相反材料；
- 论点过大，材料过少；
- 每段观点与中心论点无关。

### 7.3 解释强度分级

| 材料情况 | 合适表达 | 不合适表达 |
|---|---|---|
| 多处文本结构反复支持 | 文本反复构造了…… | 作者必然意图…… |
| 单一段落支持 | 该段显示 / 暗示 | 整部作品证明 |
| 史料部分支持 | 该史料提示一种可能解释 | 事实就是 |
| 理论视角解释 | 可从……角度理解 | 这就是文本真实意义 |
| 语境相关 | 在该语境下可以理解为 | 在所有时代都如此 |

### 7.4 引用与解释标准

每次引用材料后，作者都应完成至少一个动作：

```text
1. 解释关键词；
2. 分析修辞或形式；
3. 说明语境意义；
4. 与前文材料比较；
5. 回应替代读法；
6. 连接子论点；
7. 推进中心论点。
```

Agent 判断：

```text
如果引用之后没有解释，这段通常只是材料堆砌。
```

## 8. 文献综述写法

### 8.1 文献综述的真正功能

人文学科文献综述应构建“争论地图”，包括：

```text
对象如何被解释？
解释之间的分歧在哪里？
已有研究忽略了什么材料、形式、语境或概念？
本文如何介入并推进争论？
```

### 8.2 从研究史到争论地图

弱研究史：

```text
20 世纪 80 年代，A 研究了……
90 年代，B 提出……
近年来，C 关注……
```

强争论地图：

```text
关于该文本的研究主要在两种解释之间展开：一种强调其政治寓言性质，另一种强调其私人经验书写。前者能够解释文本中的历史象征，后者则揭示人物情感结构。然而，两种读法都较少关注叙事形式如何同时组织政治和私人经验。本文认为，正是在叙事时间的反复中断处，文本将二者联系起来。
```

### 8.3 文献综述段落模板

```text
已有研究通常将 X 理解为 A。该读法的贡献在于……，但它难以解释……。另一类研究转向 B，强调……。本文接受 B 对……的关注，但进一步指出……。因此，本文将从……角度重新考察 X。
```

### 8.4 常见错误

- 文献综述太长；
- 没有区分主要争论；
- 对已有研究只做价值判断；
- 不进入最相关争论；
- 不说明本文与已有研究关系；
- 把“没人研究过”当作唯一缺口；
- 用脚注塞满研究史，正文没有争论。

### Agent 质量判断

```text
1. 文献综述是否形成争论地图？
2. 是否概括解释路径而非逐篇罗列？
3. 是否说明本文介入点？
4. 是否处理最强对立观点？
5. 是否为中心论点铺垫？
```

## 9. 结果、分析或论证主体的组织方式

人文学科论文的主体不是 Results，而是 Argument。它应围绕中心论点分阶段推进。

### 9.1 论证链设计

Agent 可帮助作者把论文压缩为：

```text
主论点：
……

子论点 1：
……
使用材料：
……
作用：
……

子论点 2：
……
使用材料：
……
作用：
……

子论点 3：
……
使用材料：
……
作用：
……

替代解释：
……
回应：
……
```

### 9.2 章节推进方式

常见推进方式：

#### 从局部到整体

```text
词语细节
→ 段落结构
→ 整体叙事
→ 文本意义
```

#### 从材料矛盾到解释重构

```text
旧解释无法说明的矛盾
→ 新材料 / 新细读
→ 重新解释
```

#### 从语境到文本

```text
历史语境
→ 文本形式
→ 意义变化
```

#### 从概念区分到理论回应

```text
概念混淆
→ 区分 A / B
→ 重读文本
→ 修正争论
```

#### 从反读到新论点

```text
承认既有读法
→ 指出其盲点
→ 提出更强解释
```

### 9.3 段落服务主论点检查

每段都应能回答：

```text
本段主张是什么？
使用什么材料？
解释动作是什么？
它服务哪个子论点？
它如何推进主论点？
```

如果不能回答，段落可能需要删除、合并或改写。

### 9.4 常见主体失败

| 失败模式 | 表现 | 修正 |
|---|---|---|
| 材料堆砌 | 大量引文但无解释 | 每段加 claim 和 warrant |
| 观点散乱 | 每段一个新想法 | 重建主论点和子论点 |
| 理论套话 | 理论术语替代细读 | 回到材料分析 |
| 研究史替代论证 | 正文仍在介绍文献 | 将争论压缩到前文 |
| 背景泛滥 | 历史语境压倒文本 | 只保留服务论点的语境 |
| 无反驳 | 不处理替代读法 | 增加 counter-reading |

## 10. 讨论与结论写法

### 10.1 Discussion 与 Conclusion 的区别

| 部分 | 功能 |
|---|---|
| Discussion | 说明论证对学术争论、理论或历史理解的意义 |
| Conclusion | 收束全文，重申论点、贡献和边界 |

人文学科论文不一定单独设置 Discussion，但其结尾部分必须完成“从论证到意义”的提升。

### 10.2 从材料解释到贡献

材料解释：

```text
文本中的沉默多次出现在权威话语之后。
```

子论点：

```text
沉默在叙事中中断权威话语。
```

中心论点：

```text
沉默是一种低可见度行动，而非单纯被动顺从。
```

理论贡献：

```text
这使我们能够超越“发声 / 失声”的二分，重新理解该文本中的女性行动形式。
```

### 10.3 贡献表达公式

```text
本文不是将 X 再次解释为 A，而是说明 X 如何通过 B 机制产生 C 意义。
```

```text
本文修正了既有研究中将 X 与 Y 简单对立的读法，指出二者在 Z 处发生交织。
```

```text
通过重新解释 A 史料，本文将某事件从“政策失败”问题转化为“地方解释和制度协商”问题。
```

```text
本文对某概念的辨析表明，既有研究将 A 与 B 混同，因而误读了文本中的核心张力。
```

### 10.4 边界与未来研究

边界应说明：

- 材料范围；
- 文本版本；
- 时段；
- 地域；
- 语境；
- 理论适用范围；
- 解释不处理的问题。

未来研究应从边界推出：

```text
由于本文主要分析某单一文本内部的叙事机制，未来研究可比较同一时期其他文本是否也通过沉默组织女性行动。
```

### 10.5 常见错误

- 结论只是复述正文；
- 贡献写得比论证更大；
- 不回到学术争论；
- 不说明边界；
- 用情感化语言结尾；
- 引入新材料或新理论；
- 把本文解释写成唯一正确读法。

## 11. 审稿人关注点

### 11.1 论点是否新

审稿人会问：

```text
1. 本文是否只是重复已有解释？
2. 中心论点相对既有研究新在哪里？
3. 新材料是否带来新解释？
4. 新理论视角是否真正改变了读法？
```

### 11.2 是否进入学术争论

审稿人会问：

```text
1. 作者是否了解已有研究？
2. 是否准确呈现主要争论？
3. 是否回应最相关观点？
4. 论文是否只是自说自话？
```

### 11.3 材料解读是否深入

审稿人会问：

```text
1. 是否有细读？
2. 史料是否被语境化？
3. 引文是否被分析？
4. 是否只是复述内容？
5. 是否处理材料矛盾？
```

### 11.4 语境是否充分

审稿人会问：

```text
1. 作者是否说明文本、史料或作品的历史语境？
2. 是否有版本、来源和材料问题说明？
3. 是否避免时代背景泛化？
4. 语境是否真正服务解释？
```

### 11.5 理论使用是否恰当

审稿人会问：

```text
1. 理论是否帮助解释材料？
2. 是否只是套用理论术语？
3. 是否过度理论化？
4. 理论和材料是否彼此校正？
```

### 11.6 论证结构是否清楚

审稿人会问：

```text
1. 是否有中心论点？
2. 每节是否有子论点？
3. 小节之间是否递进？
4. 每段是否服务主论点？
5. 是否存在材料堆砌？
```

### 11.7 是否回应替代解释

审稿人会问：

```text
1. 是否处理最强反读？
2. 是否公平呈现不同观点？
3. 是否忽略不利材料？
4. 是否把可争论解释写成显然事实？
```

## 12. 常见失败模式

### 12.1 没有中心论点

表现：

- 只有“本文研究……”；
- 正文材料丰富但没有统领判断；
- 结论无法回答“所以呢”。

修正：

```text
1. 将主题改为问题。
2. 将问题改为可争论论点。
3. 用一句话写出 thesis。
4. 删除不服务 thesis 的材料。
```

### 12.2 材料堆砌

表现：

- 大量引文、史料和背景；
- 作者解释很少；
- 读者不知道材料证明什么。

修正：

```text
1. 每段先写主张。
2. 引用后必须细读或解释。
3. 材料按子论点重组。
4. 删除重复材料。
```

### 12.3 观点散乱

表现：

- 每段都有一个想法；
- 小节之间没有递进；
- 论文像读书笔记合集。

修正：

```text
1. 提炼主论点。
2. 将观点分为 2—4 个子论点。
3. 为每节写推进功能。
4. 增加过渡句。
```

### 12.4 理论套话

表现：

- 理论术语很多；
- 材料分析很少；
- 任何文本都能套同一段理论。

修正：

```text
1. 减少理论名词。
2. 明确理论解决什么解释问题。
3. 回到文本或史料细节。
4. 让材料修正理论。
```

### 12.5 文献综述像研究史

表现：

- 按时间或作者罗列；
- 没有争论地图；
- 不说明本文介入点。

修正：

```text
1. 按解释路径重组。
2. 指出分歧和盲点。
3. 明确本文与已有研究关系。
```

### 12.6 语境不足或语境泛滥

表现：

- 没有必要背景，读者无法理解；
- 或背景太多，淹没论点。

修正：

```text
1. 只保留解释材料所需语境。
2. 每段语境都要连接论点。
3. 删除百科式背景。
```

### 12.7 不回应替代解释

表现：

- 把自己的读法当唯一读法；
- 不处理既有强读法；
- 忽略反证材料。

修正：

```text
1. 找出最强反读。
2. 公平呈现。
3. 用材料说明本文读法优势。
4. 必要时限定论点。
```

### 12.8 结论过大

表现：

- 单一文本推出整个时代；
- 一个史料推出宏大历史规律；
- 个案解释上升到普遍理论。

修正：

```text
1. 限定材料范围。
2. 将普遍主张改为情境化贡献。
3. 用“本文表明在……中”替代“这证明……”。
```

## 13. Agent 写作辅助策略

### 13.1 Agent 总体流程

Agent 辅助人文学科论文时，应按以下顺序诊断：

```text
1. 识别研究对象。
2. 提取解释性问题。
3. 判断是否进入学术争论。
4. 提炼 thesis statement。
5. 检查材料和语境。
6. 拆解章节子论点。
7. 检查每段是否服务主论点。
8. 检查材料是否被解释。
9. 识别替代解释。
10. 重构论证链。
11. 提炼贡献。
12. 降低过度概括。
```

### 13.2 Agent 如何提炼中心论点

可依次追问：

```text
1. 你研究的对象是什么？
2. 已有研究通常如何理解它？
3. 你认为这种理解哪里不够？
4. 你的不同解释是什么？
5. 哪些材料最能支持这个解释？
6. 如果你的论点成立，学界需要如何重新理解该对象？
```

输出模板：

```text
本文不是简单讨论【对象】中的【主题】，而是论证【对象】通过【材料机制 / 形式机制 / 历史机制】呈现了【新的解释】，从而修正了既有研究中【旧解释】的局限。
```

### 13.3 Agent 如何构建章节论证链

生成表格：

| 章节 | 子论点 | 主要材料 | 解释动作 | 如何支持主论点 |
|---|---|---|---|---|
| 第一节 |  |  |  |  |
| 第二节 |  |  |  |  |
| 第三节 |  |  |  |  |

判断：

```text
如果某节不能说明如何支持主论点，应重写、合并或删除。
```

### 13.4 Agent 如何检查每段是否服务主论点

对每段做五问：

```text
1. 本段主张是什么？
2. 本段使用什么材料？
3. 作者如何解释材料？
4. 本段支持哪个子论点？
5. 本段是否推进中心论点？
```

若无法回答，应标记为：

- 背景冗余；
- 材料堆砌；
- 理论套话；
- 论点游离；
- 过渡缺失；
- 需要合并。

### 13.5 Agent 如何改写材料堆砌段

弱段落：

```text
文本中多次出现“门”。第一处是……第二处是……第三处是……
```

改写方向：

```text
先提出解释主张：
“门”不是单纯空间物件，而是人物行动可能性的界限。

然后选择最关键的 2—3 处材料细读，说明每一次“门”的出现如何改变人物关系或叙事节奏，最后回扣中心论点。
```

### 13.6 Agent 如何检查理论套话

高风险信号：

```text
只出现理论家名字；
材料可以替换成任何文本；
理论术语没有具体解释功能；
引用理论后没有细读；
结论只是“体现了某理论”。
```

修正策略：

```text
1. 明确理论概念解决什么解释问题；
2. 将理论术语转化为可观察材料特征；
3. 用材料限制或修正理论；
4. 删除不能服务论证的理论段落。
```

### 13.7 Agent 如何帮助回应替代解释

步骤：

```text
1. 找出既有最强读法。
2. 总结它解释了什么。
3. 找出它解释不了的材料。
4. 说明本文读法如何解释更多材料。
5. 承认本文读法的边界。
```

### 13.8 Agent 如何降级过度表述

| 过强表达 | 更稳妥表达 |
|---|---|
| 这证明了整个时代的精神 | 这揭示了该文本对某种时代经验的特定组织方式 |
| 作者意图显然是 | 文本结构使这种读法成为可能 |
| 所有女性形象都体现 | 本文分析的几个女性形象共同呈现 |
| 该事件的根本原因是 | 该材料提示这一因素在事件展开中具有重要作用 |
| 这一理论完全适用 | 该理论有助于解释……，但也需要结合……语境修正 |
