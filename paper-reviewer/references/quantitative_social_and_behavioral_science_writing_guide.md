# 定量社会科学 / 行为科学论文写作指南

## 1. 适用范围

本指南适用于以**理论机制、变量测量、研究设计、统计模型和经验识别**为核心的定量社会科学 / 行为科学论文。其目标是帮助 Agent 在辅助用户撰写、重构、审稿、润色论文时，判断论文是否形成了清晰的“理论机制—经验识别”闭环。

适用领域包括但不限于：

- 心理学；
- 教育学；
- 管理学；
- 组织行为学；
- 传播学；
- 社会学；
- 政治学；
- 经济学；
- 公共管理；
- 公共政策；
- 社会心理学；
- 行为科学；
- 犯罪学；
- 人口学；
- 劳动研究；
- 社会工作研究；
- 城市与区域研究；
- 健康社会科学；
- 其他以问卷、实验、准实验、计量模型、统计模型或大样本社会数据为核心的论文。

本指南适合以下论文类型：

- 问卷调查研究；
- 横断面统计分析；
- 纵向追踪研究；
- 面板数据研究；
- 实验研究；
- 准实验研究；
- 自然实验研究；
- 差分中的差分研究；
- 断点回归研究；
- 工具变量研究；
- 匹配和倾向得分研究；
- 多层模型研究；
- 结构方程模型研究；
- 中介和调节机制研究；
- 大规模行政数据研究；
- 行为实验和在线实验；
- 教育测量和心理测量研究；
- 政策评估研究；
- 组织、市场、平台或制度数据分析研究。

这类论文的核心不是“跑出了显著结果”，而是：

> 用清晰理论机制提出可检验假设，通过合理变量测量和研究设计识别经验关系，并用统计结果回到理论贡献。

也就是说，论文必须回答：

```text
理论问题是什么？
已有理论解释哪里不足？
本文提出什么机制？
机制如何转化为假设？
假设如何转化为变量和模型？
数据和识别策略能否支持推断？
结果是否回到理论机制？
贡献是否超出“发现显著相关”？
```

## 2. 不适用范围

本指南不应机械套用于以下情况。

### 2.1 纯质性研究论文

如果论文主要依赖访谈、田野观察、文本解释、扎根理论、民族志或个案过程追踪来生产知识，应使用质性研究论文写作范式。  
定量社科论文关注变量、样本、模型和统计推断；质性研究关注意义、过程、机制生成和情境化解释。

### 2.2 纯理论或概念论文

如果论文主要依赖概念分析、理论推演、规范论证或文献整合，而没有统计数据分析，则不应套用本指南。

### 2.3 自然科学实验论文

自然科学实验论文通常通过物理、化学、生物或环境实验解释自然现象。  
定量社科和行为科学虽然也有实验，但研究对象是人、组织、制度、群体和社会行为，其核心难点通常包括：

- 概念操作化；
- 测量误差；
- 选择偏倚；
- 社会情境差异；
- 内生性；
- 伦理和干预可行性；
- 统计识别和外部效度。

### 2.4 计算 benchmark 论文

如果论文主要提出算法、模型、数据集或 benchmark，并通过 baseline、指标和排行榜评价方法，则更适合计算 / 算法 / 数据驱动型论文范式。

定量社科论文不是主要证明“模型预测性能最好”，而是通过模型估计回答社会或行为理论问题。预测模型可以作为工具，但必须服务于理论解释、因果识别、机制检验或政策评估。

### 2.5 单纯数据报告或统计年鉴式分析

如果文本只是描述比例、趋势、平均数或相关系数，而没有理论问题、机制假设、变量操作化和识别逻辑，则不构成完整定量社科论文。

### 2.6 咨询报告或政策简报

政策简报强调行动建议和结果传达；定量社科论文需要更系统地说明理论背景、研究设计、识别策略、模型假设、稳健性和学术贡献。

### 2.7 只做机器学习预测的社会数据分析

如果研究只追求预测准确率，而不解释变量、机制、效应大小、偏倚、群体差异或社会意义，则不应写成传统定量社科论文。  
若要形成社科贡献，需要说明预测结果如何服务理论、政策或行为理解。

## 3. 知识生产逻辑

定量社会科学 / 行为科学论文通过以下链条生产可信知识：

```text
社会 / 行为现象
→ 理论问题
→ 文献缺口
→ 概念定义
→ 理论机制
→ 可检验假设
→ 变量操作化
→ 数据与样本
→ 研究设计与识别策略
→ 统计模型
→ 结果解释
→ 稳健性与替代解释检验
→ 理论贡献与实践意义
```

### 3.1 这类论文依靠什么产生可信知识

| 可信性来源 | 写作中应体现什么 | Agent 判断重点 |
|---|---|---|
| 理论机制 | 为什么 X 会影响 Y，或为什么某关系在 Z 条件下变化 | 是否有机制而非只猜关系 |
| 概念定义 | 核心概念边界和维度 | 概念是否清楚 |
| 变量操作化 | 概念如何转为可测变量 | 测量是否有效 |
| 研究设计 | 实验、调查、面板、准实验、行政数据等 | 设计是否匹配问题 |
| 统计识别 | 估计关系是否可解释为关联、预测、因果或机制 | 推断强度是否合适 |
| 模型设定 | 因变量、自变量、控制变量、固定效应、误差结构 | 模型是否合理 |
| 偏倚处理 | 选择偏倚、遗漏变量、反向因果、共同方法偏差等 | 是否考虑替代解释 |
| 不确定性 | 标准误、置信区间、统计功效、稳健性 | 是否避免只看显著性 |
| 实际意义 | 效应量、边际效应、预测概率、政策含义 | 是否解释大小 |
| 理论回扣 | 结果如何支持、修正或挑战理论 | 是否有贡献闭环 |

### 3.2 定量社科论文与自然科学实验论文的区别

| 维度 | 定量社会科学 / 行为科学 | 自然科学实验论文 |
|---|---|---|
| 研究对象 | 人、组织、制度、群体、政策、行为 | 自然现象、材料、生物、物理过程 |
| 核心难点 | 概念测量、社会情境、选择偏倚、内生性 | 实验控制、测量误差、机制表征 |
| 变量性质 | 许多变量是社会构念，需要操作化 | 许多变量可直接物理测量 |
| 证据目标 | 检验理论机制、估计关系或识别因果效应 | 解释自然机制或验证实验效应 |
| 可控性 | 许多社会变量不能完全操控 | 实验控制通常更强 |
| 外部效度 | 人群、制度、文化和时间高度相关 | 条件和体系相关 |
| 常见风险 | 共同方法偏差、遗漏变量、反向因果 | 对照不足、重复不足、机制过度解释 |

### 3.3 定量社科论文与计算 benchmark 论文的区别

| 维度 | 定量社科 / 行为科学论文 | 计算 benchmark 论文 |
|---|---|---|
| 核心目标 | 解释社会 / 行为关系或评估因果效应 | 评价算法 / 模型性能 |
| 关键问题 | 理论机制是否成立 | 方法是否优于 baseline |
| 变量意义 | 变量承载理论概念 | 特征和标签服务预测任务 |
| 模型评价 | 效应量、显著性、识别、稳健性 | 指标、排行榜、泛化、效率 |
| 主要贡献 | 理论、机制、测量、因果、政策解释 | 模型、算法、数据、benchmark |
| 常见误区 | “显著”但理论弱 | “SOTA”但解释弱 |

### 3.4 理论机制与经验识别之间的闭环

强论文应形成：

```text
理论机制：
为什么工作自主性会影响员工创新行为？

假设：
工作自主性通过提升心理授权感促进创新行为。

变量：
工作自主性量表、心理授权量表、创新行为量表。

模型：
检验主效应、中介效应，并控制个体和组织层面变量。

识别：
使用多时间点数据降低共同方法偏差，控制基线创新行为，进行替代模型和稳健性检验。

结果：
自主性显著预测后续创新行为，中介效应通过心理授权成立。

讨论：
结果支持自我决定理论，并说明组织设计对创新行为的作用机制。
```

弱论文常见断裂：

```text
理论说 A 影响 B，
但变量测的是 C。

假设说存在机制，
但模型只检验相关。

论文声称因果，
但数据是横断面问卷。

结果显著，
但效应大小很小且不解释。

做了很多稳健性，
但没有回到理论贡献。
```

Agent 检查时应建立“理论—假设—变量—模型—结果”矩阵：

| 理论概念 | 假设 | 操作变量 | 模型检验 | 结果 | 是否闭环 |
|---|---|---|---|---|---|
| X | H1 | X_measure | β1 | 显著 / 不显著 | 是 / 否 |
| 机制 M | H2 | M_measure | mediation | 支持 / 不支持 | 是 / 否 |
| 边界 Z | H3 | interaction X×Z | β3 | 支持 / 不支持 | 是 / 否 |

### 3.5 定量社科中的 Claim-Evidence-Warrant

| 元素 | 含义 | 示例 |
|---|---|---|
| Claim | 理论或经验主张 | 社会支持降低职业倦怠 |
| Evidence | 统计结果 | 回归系数为 -0.24，95% CI [-0.31, -0.17] |
| Warrant | 解释桥梁 | 社会支持提供情绪和资源缓冲，降低压力消耗 |
| Boundary | 适用条件 | 该结果来自教师样本，不能直接推广到所有行业 |

Agent 检查规则：

```text
每当论文写“影响”“导致”“促进”“抑制”“机制”“调节”“中介”“显著提高”时，必须检查：
1. 理论机制在哪里？
2. 变量如何测量？
3. 模型是否检验了该主张？
4. 研究设计是否支持因果语言？
5. 是否有替代解释？
6. 效应量是否具有实际意义？
7. 结论是否限定样本和情境？
```

## 4. 典型论文结构

定量社会科学 / 行为科学论文常见结构如下：

```text
Title
Abstract
Keywords

1. Introduction
2. Theory / Literature Review
3. Hypotheses
4. Data / Sample
5. Measures / Variables
6. Empirical Strategy / Methods
7. Results
8. Robustness / Additional Analyses
9. Discussion
10. Conclusion

References
Appendices / Supplementary Materials
```

不同领域会有变体：

- 心理学和管理学常用：Introduction → Theory and Hypotheses → Method → Results → Discussion；
- 经济学和政治学常用：Introduction → Background / Theory → Data → Empirical Strategy → Results → Robustness → Conclusion；
- 教育学常用：Introduction → Literature → Conceptual Framework → Data and Measures → Analytic Strategy → Results → Discussion；
- 传播学常用：Theory → Hypotheses / Research Questions → Method → Results → Discussion；
- 社会学常用：Theory → Data → Measures → Models → Results → Discussion；
- 公共管理 / 政策评估常用：Policy Background → Research Design → Data → Empirical Strategy → Results → Robustness → Implications。

### 4.1 章节功能总览

| 章节 | 核心功能 | 不应写成 |
|---|---|---|
| Introduction | 提出社会 / 行为问题、理论缺口和贡献 | 泛泛现实背景 |
| Theory / Literature | 推导机制和假设 | 文献罗列 |
| Hypotheses | 将理论机制转为可检验命题 | 经验猜测 |
| Data / Sample | 说明数据来源、样本和代表性 | 数据说明流水账 |
| Measures / Variables | 说明概念如何操作化 | 变量名列表 |
| Empirical Strategy | 说明模型、识别、估计和推断 | 软件操作说明 |
| Results | 呈现主效应、机制、边界和检验 | 表格堆砌 |
| Robustness | 排除替代解释、检验稳健性 | 机械换模型 |
| Discussion | 回到理论贡献、实际意义和边界 | 重复结果 |
| Conclusion | 收束问题、贡献和未来研究 | 空泛总结 |

### 4.2 结构质量判断

Agent 应检查：

```text
1. Introduction 是否明确理论问题？
2. Theory 是否能推出假设？
3. Hypotheses 是否与变量和模型对应？
4. Data 是否适合回答问题？
5. Measures 是否能测量理论概念？
6. Empirical Strategy 是否说明识别逻辑？
7. Results 是否按假设和机制组织？
8. Robustness 是否针对关键威胁？
9. Discussion 是否回到理论贡献？
10. Conclusion 是否避免过度因果或泛化？
```

## 5. 各章节写作指南

## 5.1 标题 Title

### 写作目标

标题应准确传达研究对象、理论关系、数据类型、研究设计或核心贡献。

### 常见标题类型

| 类型 | 结构 | 示例形式 |
|---|---|---|
| 关系型 | X and Y: Evidence from Z | 变量关系 |
| 机制型 | How X Shapes Y through M | 中介机制 |
| 边界型 | When X Matters: The Moderating Role of Z | 调节机制 |
| 因果识别型 | The Effect of X on Y: Evidence from a Natural Experiment | 政策评估 |
| 问题型 | Does X Increase Y? | 清晰研究问题 |
| 场景型 | X in Y Context | 特定制度 / 人群 |
| 方法型 | A Panel Study / Survey Experiment / Field Experiment of X | 研究设计 |

### 常见错误

- 标题只写主题，不写关系；
- 标题暗示因果，但研究设计只能支持相关；
- 标题过度使用“影响因素研究”；
- 标题太泛，如“大学生心理健康研究”；
- 标题承诺机制，但正文没有中介或机制检验；
- 标题没有体现样本或情境边界。

### Agent 质量判断

```text
1. 标题是否体现核心变量或理论关系？
2. 是否体现研究设计或数据特征？
3. 是否避免因果过度声称？
4. 是否与实际模型和结果一致？
5. 是否比泛泛主题更具体？
```

---

## 5.2 摘要 Abstract

### 写作目标

摘要应压缩完整研究逻辑：

```text
社会 / 行为问题
→ 理论缺口
→ 数据与方法
→ 主要发现
→ 理论贡献
→ 实际意义和边界
```

### 推荐结构

```text
1. 背景句：指出社会 / 行为现象或理论问题。
2. 缺口句：说明已有研究不足。
3. 理论句：概括本文机制或假设。
4. 方法句：说明数据、样本和模型。
5. 结果句：报告核心效应和机制。
6. 贡献句：说明对理论或实践的贡献。
7. 边界句：说明适用范围或限制。
```

### 示例骨架

```text
组织授权被认为能够促进员工创新行为，但其作用机制和边界条件仍存在争议。既有研究多关注授权的直接效果，较少区分心理授权与资源可得性的中介作用。本文基于自我决定理论，提出组织授权通过提升心理授权感促进创新行为，并且该关系在高任务不确定性情境下更强。基于三波问卷数据和多层回归模型，研究发现……。结果表明……。本文贡献在于……。由于样本来自……，结论应限于……情境解释。
```

### 常见错误

- 摘要只写背景和方法；
- 不说明理论机制；
- 只说“结果显著”；
- 不报告样本和数据来源；
- 不说明研究设计；
- 把相关写成因果；
- 不说明贡献；
- 不说明边界。

### Agent 质量判断

```text
1. 摘要是否明确理论问题？
2. 是否说明数据和研究设计？
3. 是否概括核心假设或机制？
4. 是否报告主要结果而非只说显著？
5. 是否说明贡献？
6. 是否避免过度因果和泛化？
```

## 5.3 Introduction 引言

### 写作目标

Introduction 应回答：

```text
这个社会 / 行为问题为什么重要？
已有理论解释哪里不足？
本文提出什么新机制或新识别？
本文数据和方法如何回答？
贡献是什么？
```

### 推荐组织方式

```text
1. 现象或问题：具体社会 / 行为现象。
2. 理论重要性：该现象连接哪个理论争议。
3. 已有研究：已知什么。
4. 缺口：未解释什么，或已有结论为何不一致。
5. 本文问题：提出研究问题。
6. 理论机制：简要说明本文解释。
7. 数据与设计：说明如何检验。
8. 贡献：理论、方法、数据、识别、实践贡献。
9. 文章结构。
```

### 好的引言写法

弱写法：

```text
随着社会发展，员工创新越来越重要，已有很多研究关注这一问题。
```

强写法：

```text
尽管组织授权常被视为激发员工创新的重要管理实践，但现有研究对其作用机制仍存在分歧。一类研究强调授权带来的自主性，另一类研究则指出授权可能增加角色压力。两种解释的差异表明，授权并非在所有情境下都产生相同效果，关键问题在于员工如何解释和使用授权所带来的决策空间。
```

### 理论缺口写法

弱缺口：

```text
目前关于 X 和 Y 的研究较少。
```

强缺口：

```text
已有研究已经证明 X 与 Y 相关，但较少解释这种关系何时通过资源机制发挥作用、何时通过身份认同机制发挥作用。因此，现有理论难以解释不同研究中效应方向和大小不一致的现象。
```

### 贡献写法

弱贡献：

```text
本文丰富了相关研究，并具有实践意义。
```

强贡献：

```text
本文贡献有三点。第一，本文将 X 理论与 Y 理论结合，提出一个解释 A 与 B 关系的双机制框架。第二，本文使用三波追踪数据，将预测变量、中介变量和结果变量分时测量，以降低共同方法偏差。第三，本文通过边际效应和异质性分析说明该机制主要发生在……情境中，从而限定了已有理论的适用范围。
```

### 常见错误

- 现实背景很长，理论问题很弱；
- 引言只说变量关系，没有理论争议；
- 缺口只是“研究少”；
- 贡献只写“丰富理论”；
- 方法贡献和理论贡献混淆；
- 研究问题与模型不匹配；
- 引言声称因果，但数据不支持；
- 没有说明为什么当前样本或情境重要。

### Agent 质量判断

```text
1. 是否提出具体社会 / 行为问题？
2. 是否连接理论争议或理论缺口？
3. 是否说明已有研究不足？
4. 是否提出清晰研究问题？
5. 是否简要说明数据和设计？
6. 是否声明具体贡献？
7. 是否避免泛泛现实背景？
```

## 5.4 Theory / Literature 理论与文献

### 写作目标

Theory 部分不是文献综述清单，而是从已有理论中推导机制和假设。

它应回答：

```text
核心概念是什么？
已有理论如何解释它们之间的关系？
为什么本文提出的机制合理？
为什么某些条件会改变这种关系？
假设如何从理论推导出来？
```

### 推荐组织方式

```text
1. 概念定义；
2. 理论基础；
3. 已有研究解释路径；
4. 本文机制推导；
5. 主效应假设；
6. 中介 / 调节 / 异质性假设；
7. 理论模型图，如适用。
```

### 理论推导结构

每个假设前的理论段落可采用：

```text
概念定义
→ 理论前提
→ 机制链条
→ 替代解释或边界
→ 假设
```

示例：

```text
自我决定理论认为，个体在感到自主、胜任和关联时更可能产生内在动机。工作自主性提供了对任务方式和节奏的控制权，因此能够提高员工对自身工作能力和责任的感知。较高的心理授权感进一步促使员工尝试新方法并承担创新风险。因此，本文提出：
H1：工作自主性与员工创新行为正相关。
H2：心理授权感中介工作自主性与员工创新行为之间的关系。
```

### 文献组织方式

#### 方式一：理论流派组织

```text
资源理论
身份理论
制度理论
本文整合机制
```

#### 方式二：关系路径组织

```text
X 对 Y 的直接影响
X 通过 M 影响 Y
Z 调节 X 与 Y 的关系
```

#### 方式三：争议组织

```text
研究认为 X 促进 Y
研究认为 X 抑制 Y
本文解释何时促进、何时抑制
```

#### 方式四：层次组织

```text
个体层机制
团队层机制
组织 / 制度层机制
跨层假设
```

### 常见错误

- 按作者逐篇罗列；
- 理论只是引用名词，没有推导；
- 假设突然出现；
- 文献综述与变量测量不对应；
- 概念定义前后不一致；
- 将经验发现当作理论机制；
- 不处理相反研究；
- 理论机制和统计模型脱节。

### Agent 质量判断

```text
1. Theory 是否定义核心概念？
2. 是否有明确理论基础？
3. 是否解释机制链条？
4. 假设是否自然推出？
5. 是否处理竞争解释？
6. 是否避免文献罗列？
7. 是否与变量和模型对应？
```

## 5.5 Hypotheses 假设

### 写作目标

假设是理论机制的可检验表达。它应清楚说明变量之间的预期关系、方向、条件或机制。

### 常见假设类型

| 类型 | 形式 | 示例 |
|---|---|---|
| 主效应 | X 影响 Y | H1：X 与 Y 正相关 |
| 中介效应 | X 通过 M 影响 Y | H2：M 中介 X 与 Y 的关系 |
| 调节效应 | Z 改变 X-Y 关系 | H3：Z 正向调节 X 与 Y 的关系 |
| 调节中介 | Z 改变间接效应 | H4：X 通过 M 影响 Y 的间接效应在 Z 高时更强 |
| 异质性 | 效应在群体间不同 | H5：X 对 Y 的影响在低资源群体中更强 |
| 非线性 | 关系呈倒 U 或阈值 | H6：X 与 Y 呈倒 U 型关系 |
| 因果假设 | X 导致 Y | 需要强设计支撑 |
| 研究问题 | 不预设方向 | RQ：X 与 Y 的关系是否因 Z 而变化？ |

### 好的假设写法

弱写法：

```text
H1：X 对 Y 有显著影响。
```

强写法：

```text
H1：在控制个体基线特征后，工作自主性与后续创新行为正相关。
```

弱写法：

```text
H2：M 具有中介作用。
```

强写法：

```text
H2：心理授权感中介工作自主性与员工创新行为之间的正向关系。
```

弱写法：

```text
H3：Z 调节 X 和 Y。
```

强写法：

```text
H3：任务不确定性正向调节工作自主性与心理授权感之间的关系；当任务不确定性较高时，该关系更强。
```

### 假设写作原则

```text
1. 明确变量；
2. 明确方向；
3. 明确作用位置；
4. 明确层级；
5. 明确时间顺序，如适用；
6. 避免“显著影响”这种模糊表达；
7. 假设数量与理论和数据能力匹配；
8. 不要为每个控制变量写假设。
```

### 常见错误

- 假设没有理论推导；
- 假设写成统计命令；
- 方向不清；
- 中介、调节位置不清；
- 假设太多，像探索性分析；
- 假设和模型不对应；
- 数据无法检验假设；
- 横断面数据写强因果假设。

### Agent 质量判断

```text
1. 每个假设是否来自前文理论？
2. 每个假设是否能被变量和模型检验？
3. 假设是否明确方向和关系？
4. 中介 / 调节 / 异质性是否说清作用位置？
5. 假设数量是否合理？
6. 是否避免把探索性分析伪装成假设？
```

## 5.6 Data / Sample 数据与样本

### 写作目标

Data / Sample 部分说明经验基础是否适合回答研究问题。

它应回答：

```text
数据来自哪里？
样本如何产生？
研究对象是谁？
样本是否代表目标总体？
是否存在选择偏倚？
样本量是否足以支持模型？
```

### 应包含内容

- 数据来源；
- 采集方式；
- 时间范围；
- 地理范围；
- 研究对象；
- 抽样方法；
- 回收率或匹配率；
- 纳入和排除标准；
- 样本量；
- 缺失情况；
- 样本权重，如适用；
- 与总体的比较；
- 伦理审批或知情同意，如涉及人类受试者；
- 数据使用许可。

### 问卷数据写法

应说明：

```text
1. 问卷设计来源；
2. 预测试；
3. 抽样框；
4. 发放方式；
5. 回收率；
6. 有效样本标准；
7. 时间点；
8. 匿名性和伦理；
9. 共同方法偏差控制。
```

### 行政数据 / 大样本数据写法

应说明：

```text
1. 数据提供机构；
2. 数据覆盖范围；
3. 单位：个体、学校、企业、地区、国家等；
4. 时间跨度；
5. 匹配规则；
6. 清洗规则；
7. 缺失和异常处理；
8. 变量构造；
9. 数据限制。
```

### 实验样本写法

应说明：

```text
1. 招募方式；
2. 随机分配；
3. 实验条件；
4. 排除规则；
5. 操纵检验；
6. 注意力检查；
7. 样本量 / 统计功效；
8. 伦理和补偿。
```

### 常见错误

- 数据来源不清；
- 样本筛选过程不透明；
- 没有说明排除标准；
- 样本与理论目标总体不一致；
- 样本量不足但模型复杂；
- 不报告缺失值；
- 问卷研究不报告回收率；
- 实验研究不说明随机分配；
- 将便利样本结论推广到全体人群。

### Agent 质量判断

```text
1. 样本来源是否清楚？
2. 抽样和筛选过程是否透明？
3. 样本是否适合研究问题？
4. 是否报告样本量和缺失？
5. 是否讨论代表性和选择偏倚？
6. 是否说明伦理和数据许可？
```

## 5.7 Measures / Variables 测量与变量

### 写作目标

Measures / Variables 部分说明理论概念如何被操作化。定量社科论文的一个核心风险是：

```text
理论概念很漂亮，但变量测量不对应。
```

因此本节必须回答：

```text
核心概念是什么？
用什么变量测量？
变量来源是什么？
测量是否可靠、有效？
变量如何编码？
控制变量为什么选择？
```

### 应包含内容

- 因变量；
- 自变量；
- 中介变量；
- 调节变量；
- 控制变量；
- 变量定义；
- 量表来源；
- 题项示例；
- 信度；
- 效度；
- 标准化或转换；
- 缺失值处理；
- 描述统计；
- 相关矩阵；
- 多重共线性检查，如适用。

### 概念与操作化

推荐表格：

| 理论概念 | 操作变量 | 数据来源 | 测量方式 | 取值范围 | 预期方向 |
|---|---|---|---|---|---|
| 工作自主性 | autonomy_scale | 问卷 | 5 项 Likert | 1-5 | + |
| 创新行为 | innovation_behavior | 主管评分 | 6 项量表 | 1-7 | 因变量 |
| 任务不确定性 | task_uncertainty | 问卷 | 4 项量表 | 1-5 | 调节 |

### 量表测量

心理学、管理学、教育学等使用量表时应说明：

- 量表来源；
- 翻译和回译，如适用；
- 题项数量；
- 样例题项；
- 量尺；
- Cronbach's alpha 或其他信度指标；
- 验证性因子分析，如适用；
- 聚合指标，如团队层变量；
- 测量等值性，如跨群体比较适用。

### 操作变量写法

弱写法：

```text
本文用问卷测量领导风格。
```

强写法：

```text
变革型领导采用 Bass 和 Avolio 量表中的 8 个题项测量，受访者使用 1 至 5 分 Likert 量表评价直接主管行为。样例题项包括……。该量表在本样本中的 Cronbach's alpha 为……。
```

### 控制变量写法

控制变量不应机械堆砌，应说明选择依据。

弱写法：

```text
本文控制了性别、年龄、学历、收入等变量。
```

强写法：

```text
本文控制年龄、性别、教育程度和基线收入，因为这些变量可能同时影响数字技能培训参与概率和后续就业结果。
```

### 常见错误

- 概念定义和变量不一致；
- 变量名列表化；
- 量表来源不清；
- 不报告信度；
- 不报告描述统计；
- 控制变量选择随意；
- 控制了中介变量或碰撞变量；
- 将单题指标当作复杂概念但不解释；
- 跨文化或跨群体研究不检验测量等值性；
- 对高度偏态变量不处理。

### Agent 质量判断

```text
1. 每个理论概念是否有对应变量？
2. 变量是否真的测量该概念？
3. 测量来源是否可靠？
4. 是否报告信度和效度？
5. 控制变量是否有理论和识别依据？
6. 是否存在不当控制？
7. 是否说明变量编码和取值？
```

## 5.8 Empirical Strategy / Methods 经验策略与模型

### 写作目标

Empirical Strategy 是定量社科论文的识别核心。它应回答：

```text
本文估计的是什么关系？
模型为什么这样设定？
研究设计是否支持因果推断？
主要识别威胁是什么？
作者如何处理内生性、混杂和偏倚？
标准误如何处理？
```

### 常见模型和设计

| 类型 | 适用问题 | 写作重点 |
|---|---|---|
| OLS / Logit / Probit | 关联分析、基础模型 | 因变量类型、系数解释 |
| 多层模型 | 个体嵌套于班级、组织、地区 | 层级结构、随机效应 |
| 固定效应模型 | 面板数据、不可观测稳定差异 | 单位 FE、时间 FE |
| 差分中的差分 | 政策或处理前后对照 | 平行趋势、处理组和对照组 |
| 断点回归 | 阈值规则 | 操纵检验、带宽 |
| 工具变量 | 内生性处理 | 相关性、排除限制 |
| 倾向得分 / 匹配 | 选择偏倚 | 平衡性、重叠性 |
| 实验模型 | 随机处理效应 | 随机化、操纵检验 |
| 中介模型 | 机制路径 | 时间顺序、因果假设 |
| 调节模型 | 边界条件 | 交互项、边际效应 |
| 结构方程模型 | 潜变量和路径 | 测量模型、拟合指标 |
| 事件研究 | 政策动态效应 | 预趋势、动态效应 |
| 生存分析 | 时间到事件 | 删失、风险假设 |

### 模型设定写法

应说明：

```text
1. 因变量；
2. 核心自变量；
3. 控制变量；
4. 固定效应；
5. 标准误；
6. 权重；
7. 样本限制；
8. 估计目标；
9. 系数如何解释；
10. 识别假设。
```

示例：

```text
本文估计以下模型：
Y_it = βX_it + γZ_it + μ_i + λ_t + ε_it。
其中 Y_it 表示个体 i 在时间 t 的就业状态，X_it 表示是否参加培训，Z_it 为时间变化控制变量，μ_i 控制个体不随时间变化的不可观测特征，λ_t 控制共同时间冲击。标准误聚类在个体层面。
```

### 因果识别写法

如果论文声称因果，必须说明识别逻辑：

```text
本文识别策略依赖于……
该假设意味着……
我们通过以下方式检验其合理性……
仍可能存在……
```

#### DID 写作重点

```text
1. 政策或处理发生时间；
2. 处理组和对照组；
3. 平行趋势假设；
4. 事件研究图；
5. 是否存在提前反应；
6. 是否存在处理异质性；
7. 标准误聚类；
8. 政策同时发生的混杂。
```

#### IV 写作重点

```text
1. 工具变量与内生变量相关；
2. 工具变量只通过内生变量影响结果；
3. 第一阶段强度；
4. 排除限制的理论解释；
5. 弱工具风险；
6. LATE 解释；
7. 过度识别检验，如适用。
```

#### RDD 写作重点

```text
1. 运行变量；
2. 截断点；
3. 带宽选择；
4. 截断点附近可比性；
5. 操纵检验；
6. 协变量平衡；
7. 图形展示；
8. 局部效应解释。
```

#### 实验写作重点

```text
1. 随机分配过程；
2. 处理条件；
3. 操纵检验；
4. 注意力检查；
5. 样本量和功效；
6. 排除规则是否预设；
7. 伦理审批；
8. 实验材料和流程可复现。
```

### 内生性处理

常见内生性来源：

- 遗漏变量；
- 反向因果；
- 选择偏倚；
- 测量误差；
- 同时性；
- 样本选择；
- 政策选择性实施；
- 网络干扰或溢出效应。

写作中应说明：

```text
1. 主要内生性威胁是什么；
2. 研究设计如何缓解；
3. 仍有哪些无法排除；
4. 通过何种稳健性或敏感性分析检查。
```

### 常见错误

- 只写回归方程，不写识别逻辑；
- 控制变量越多越好；
- 声称因果但没有设计；
- 不解释固定效应；
- 标准误层级错误；
- 中介分析缺少时间顺序；
- DID 不检验平行趋势；
- IV 不解释排除限制；
- RDD 不做操纵检验；
- 将机器学习预测结果解释为因果效应。

### Agent 质量判断

```text
1. 模型是否匹配因变量类型？
2. 研究设计是否支持作者推断？
3. 是否说明识别假设？
4. 是否处理内生性威胁？
5. 控制变量选择是否合理？
6. 标准误和聚类是否合适？
7. 是否区分关联、预测和因果？
```

## 5.9 Results 结果

### 写作目标

Results 部分应以假设和理论机制为主线呈现统计结果，而不是按软件输出顺序堆表。

推荐顺序：

```text
描述统计
→ 相关矩阵 / 平衡性
→ 主效应
→ 中介 / 调节 / 机制
→ 异质性
→ 稳健性
→ 附加分析
```

### 主效应写法

弱写法：

```text
X 的系数显著为正，说明 H1 得到验证。
```

强写法：

```text
模型 2 显示，工作自主性的系数为 0.18，95% CI [0.07, 0.29]，表明在控制个体和组织特征后，工作自主性越高，员工创新行为越高。该结果支持 H1。按标准化系数解释，自主性每增加一个标准差，创新行为约增加 0.18 个标准差，效应大小中等。
```

### 调节效应写法

调节效应不能只看交互项显著，应解释边际效应。

应报告：

- 交互项系数；
- 边际效应图；
- 简单斜率；
- Johnson-Neyman 区间，如适用；
- 高低调节变量下的效应；
- 理论解释。

弱写法：

```text
交互项显著，因此 H3 成立。
```

强写法：

```text
交互项 X×Z 为正且显著。边际效应图显示，当任务不确定性高于样本均值时，工作自主性与创新行为的关系明显增强；在低任务不确定性下，该关系接近零。这与 H3 关于任务不确定性强化自主性作用的预测一致。
```

### 中介效应写法

中介分析应谨慎，尤其在横断面数据中。

应报告：

- X → M；
- M → Y；
- 间接效应；
- bootstrap 置信区间；
- 直接效应；
- 时间顺序；
- 替代中介；
- 因果假设。

弱写法：

```text
M 的中介效应显著，说明 X 通过 M 影响 Y。
```

强写法：

```text
bootstrap 结果显示，X 通过 M 的间接效应为……，95% CI 不包含 0。由于 X、M 和 Y 来自三波测量，该时间顺序与本文机制一致。尽管如此，仍不能完全排除未测量混杂对中介路径的影响。
```

### 异质性分析写法

异质性分析应回应理论边界，而不是随意分组。

应说明：

```text
1. 为什么该群体或情境重要；
2. 异质性是否预设；
3. 分组是否合理；
4. 是否检验组间差异；
5. 是否避免只比较组内显著性。
```

### 结果表格写法

主表应清楚：

- 模型编号；
- 因变量；
- 核心自变量；
- 控制变量；
- 固定效应；
- 样本量；
- R² / pseudo R²；
- 标准误类型；
- 显著性标记；
- 估计方法。

正文不应逐项复述表格，而应解释与假设相关的结果。

### 常见错误

- 只说显著不显著；
- 不解释效应量；
- 不解释实际意义；
- 把控制变量结果当重点；
- 不报告置信区间；
- 交互项不画边际效应；
- 中介分析无理论或时间顺序；
- 异质性分析像数据挖掘；
- 结果与假设顺序不一致；
- 不讨论不支持的假设。

### Agent 质量判断

```text
1. Results 是否按假设和机制组织？
2. 是否报告系数、标准误 / CI、效应大小？
3. 是否解释实际意义？
4. 调节效应是否有边际效应解释？
5. 中介效应是否有间接效应和时间顺序说明？
6. 异质性是否理论驱动？
7. 是否处理不支持或反向结果？
```

## 5.10 Robustness / Additional Analyses 稳健性与补充分析

### 写作目标

Robustness 部分不是机械更换模型，而是回应关键威胁：

```text
结果是否依赖某个测量方式？
是否依赖某个模型设定？
是否由极端值驱动？
是否存在替代解释？
是否受到遗漏变量、选择偏倚或共同方法偏差影响？
```

### 常见稳健性检验

| 威胁 | 检验方式 |
|---|---|
| 变量测量争议 | 替代变量、不同编码 |
| 模型设定 | 替代模型、非线性、固定效应 |
| 极端值 | winsorize、删除极端值 |
| 样本选择 | 更换样本、重加权、匹配 |
| 混杂 | 增加控制、敏感性分析 |
| 反向因果 | 滞后变量、面板模型、交叉滞后 |
| 共同方法偏差 | 多源数据、分时测量、统计检验 |
| 政策评估威胁 | placebo、事件研究、替代对照组 |
| 标准误问题 | 聚类、稳健标准误、bootstrap |
| 多重检验 | FDR、预设分析、谨慎解释 |

### 好的稳健性写法

弱写法：

```text
我们进行了多种稳健性检验，结果基本一致。
```

强写法：

```text
为检验结果是否依赖创新行为的测量方式，本文使用主管评分替代员工自评作为因变量。结果显示，工作自主性的系数方向和大小与主分析相近。该检验降低了共同方法偏差解释的可能性。
```

### Placebo / Falsification 写法

```text
如果本文识别策略有效，X 不应影响理论上无关的结果 Y_placebo。结果显示 X 对 Y_placebo 的系数接近零且不显著，支持本文主结果不太可能由共同趋势驱动。
```

### 常见错误

- 稳健性检验不针对任何威胁；
- 只报告“结果一致”；
- 不展示关键表格或附录；
- 稳健性结果与主结果不同但不解释；
- 使用过多探索性模型制造显著；
- 把稳健性检验写成主结果替代；
- 不区分预设与事后分析。

### Agent 质量判断

```text
1. 稳健性是否针对主要识别威胁？
2. 每个稳健性检验是否有明确目的？
3. 是否报告关键结果？
4. 不一致结果是否解释？
5. 是否避免机械换模型？
6. 是否回扣理论和识别？
```

## 5.11 Discussion 讨论

### 写作目标

Discussion 应从统计结果回到理论机制、实际意义和边界条件。

推荐结构：

```text
1. 概括主要发现；
2. 解释理论意义；
3. 说明机制和边界条件；
4. 讨论实际意义；
5. 说明局限性；
6. 提出未来研究。
```

### 理论贡献写法

弱写法：

```text
本研究丰富了 X 理论。
```

强写法：

```text
本文结果表明，X 对 Y 的作用并非单一直接效应，而是依赖于 Z 条件下的 M 机制。这一发现修正了既有研究将 X 视为稳定正向因素的假设，并说明 X 理论需要进一步考虑情境约束。
```

### 实际意义写法

弱写法：

```text
本研究对管理实践有重要启示。
```

强写法：

```text
结果显示，工作自主性只有在任务不确定性较高时才明显促进创新行为。因此，组织在推广授权实践时，不应将自主性视为普遍有效的管理工具，而应结合任务类型和员工资源支持进行设计。
```

### 局限性写法

局限性应具体：

- 横断面数据无法强因果；
- 样本来自单一地区或行业；
- 变量自陈可能有共同方法偏差；
- 未测量关键混杂；
- 量表跨文化适用性有限；
- 政策效应可能受其他同期政策影响；
- 机制变量只是代理指标；
- 外部效度有限。

弱写法：

```text
本研究仍有不足，未来可以扩大样本。
```

强写法：

```text
本研究使用三波问卷数据降低了共同方法偏差，但仍不能完全排除未测量个体特质同时影响自主性感知和创新行为的可能。未来研究可结合组织制度变动或现场实验进一步识别因果效应。
```

### 常见错误

- Discussion 重复 Results；
- 理论贡献空泛；
- 不讨论不支持假设；
- 实际意义超过数据；
- 局限性套路化；
- 不说明因果识别限制；
- 不讨论测量误差；
- 将统计显著写成政策建议。

### Agent 质量判断

```text
1. Discussion 是否回到理论机制？
2. 是否说明结果如何支持 / 修正理论？
3. 是否解释效应大小和实际意义？
4. 是否讨论不支持的假设？
5. 是否具体说明局限？
6. 是否避免过度政策或管理建议？
```

## 5.12 Conclusion 结论

### 写作目标

Conclusion 应简洁回答：

```text
研究了什么问题？
使用什么数据和设计？
发现了什么？
对理论有什么贡献？
适用边界是什么？
```

### 推荐结构

```text
本文围绕……问题，基于……理论，提出……假设。
利用……数据和……模型，本文发现……
这一发现表明……
本文贡献在于……
由于……限制，结论应限于……
未来研究可……
```

### 常见错误

- 只重复摘要；
- 不回到理论问题；
- 不说明边界；
- 将相关写成因果；
- 将单一样本推广过大；
- 引入新结果；
- 用口号式政策建议结尾。

### Agent 质量判断

```text
1. 是否回答研究问题？
2. 是否概括核心发现而非罗列系数？
3. 是否说明理论贡献？
4. 是否限定设计和样本边界？
5. 是否避免过度因果和泛化？
```

## 6. 常见贡献类型

定量社会科学 / 行为科学论文的贡献通常不是“发现显著结果”，而是以下类型之一或组合。

### 6.1 理论机制贡献

提出或检验新的机制链条。

示例：

```text
本文表明，数字监控并非直接降低员工满意度，而是通过降低感知自主性产生负向影响。
```

质量要求：

- 机制有理论来源；
- 中介变量测量合理；
- 时间顺序或设计支持机制；
- 处理替代机制。

### 6.2 边界条件贡献

说明某理论关系在何种条件下更强、更弱或反向。

示例：

```text
本文发现，组织授权对创新行为的正向作用主要存在于高任务不确定性情境中。
```

质量要求：

- 调节变量有理论意义；
- 交互项和边际效应解释清楚；
- 不是事后数据分组。

### 6.3 因果识别贡献

通过更强研究设计识别某政策、制度或行为变量的影响。

示例：

```text
本文利用政策分阶段实施构造差分中的差分设计，估计数字政务改革对企业办事成本的影响。
```

质量要求：

- 识别假设清楚；
- 检验平行趋势或相应假设；
- 稳健性充分；
- 结论限定于识别范围。

### 6.4 测量贡献

提出、验证或改进社会 / 行为构念的测量。

示例：

```text
本文开发了平台劳动者算法控制感知量表，并验证其信度、结构效度和预测效度。
```

质量要求：

- 概念清楚；
- 题项生成合理；
- 信效度充分；
- 跨样本验证；
- 与相近概念区分。

### 6.5 数据贡献

使用新的数据源、长时段数据、跨层数据、匹配数据或高质量调查。

示例：

```text
本文将企业登记数据与员工问卷数据匹配，检验组织制度与个体态度之间的跨层关系。
```

质量要求：

- 数据新颖性服务理论问题；
- 匹配和清洗透明；
- 样本选择和代表性讨论充分。

### 6.6 异质性贡献

揭示效应在不同群体、组织、地区或制度条件下的差异。

示例：

```text
政策效果主要集中在资源约束较强的中小企业，而对大型企业影响有限。
```

质量要求：

- 异质性有理论预期；
- 组间差异正式检验；
- 避免过度数据挖掘。

### 6.7 方法贡献

将更合适的计量、实验或测量方法引入某研究问题。

示例：

```text
本文通过实验 vignette 设计区分能力信号和身份信号对招聘评价的影响。
```

质量要求：

- 方法选择解决已有识别或测量问题；
- 方法服务理论贡献；
- 不是仅为了技术复杂。

### 6.8 实证反驳贡献

发现与主流理论预期不同的结果，并提出解释。

示例：

```text
与资源依赖理论的单向预测不同，本文发现外部资源支持在高监管压力下反而降低组织创新。
```

质量要求：

- 反常结果稳健；
- 有理论解释；
- 排除测量或模型错误；
- 说明如何修正理论。

## 7. 证据与论证标准

## 7.1 强证据的特征

定量社科论文中的强证据通常具有以下特征：

```text
1. 理论机制清楚；
2. 假设可由理论推出；
3. 变量操作化与概念一致；
4. 数据适合研究问题；
5. 模型设定合理；
6. 识别假设透明；
7. 关键偏倚和内生性被处理；
8. 效应量和不确定性完整报告；
9. 稳健性检验针对主要威胁；
10. 结果回到理论贡献；
11. 结论不超过设计支持范围。
```

### 7.2 弱证据的特征

弱证据常表现为：

- 只有显著系数，没有理论机制；
- 变量测量与概念不匹配；
- 横断面问卷强行做因果；
- 控制变量随意堆砌；
- 没有解释效应大小；
- 样本选择不透明；
- 共同方法偏差严重；
- 稳健性只是换模型；
- 不处理内生性；
- 不报告缺失值；
- 中介分析没有时间顺序；
- 调节分析只看交互项，不解释边际效应；
- 异质性分析事后挑选；
- Discussion 不回到理论。

### 7.3 关联、预测、因果和机制的区分

| 推断类型 | 可以说什么 | 需要什么证据 |
|---|---|---|
| 描述 | X 的分布、趋势或差异 | 代表性数据和测量 |
| 关联 | X 与 Y 相关 | 统计相关和控制变量 |
| 预测 | X 有助于预测 Y | 训练 / 验证、预测性能 |
| 因果 | X 导致 Y | 随机、准实验、强识别假设 |
| 机制 | X 通过 M 影响 Y | 理论、时间顺序、中介证据 |
| 边界 | X-Y 关系因 Z 变化 | 调节理论和交互检验 |

Agent 判断规则：

```text
研究设计支持什么推断，论文就只能使用相应强度的语言。
```

### 7.4 效应量和实际意义

定量社科论文不应只报告显著性。应解释：

- 标准化系数；
- 边际效应；
- 预测概率变化；
- 均值差；
- 百分比变化；
- 相对于基线的变化；
- 政策或管理意义；
- 最小可感知差异，如适用；
- 经济显著性或实质显著性。

弱写法：

```text
X 的系数在 1% 水平显著。
```

强写法：

```text
X 增加一个标准差时，Y 增加 0.12 个标准差。相对于样本均值，这约等于……的变化，说明效应虽统计显著，但实际幅度较小。
```

### 7.5 统计显著性和不确定性

应报告：

- 系数；
- 标准误；
- 置信区间；
- P 值，如适用；
- 样本量；
- 模型设定；
- 标准误聚类层级；
- 多重比较处理；
- 统计功效，如适用；
- 不显著结果的解释。

Agent 应提醒：

```text
不显著不等于没有效应。
置信区间宽说明不确定性大。
统计显著不等于理论重要或实际重要。
```

## 8. 文献综述写法

### 8.1 文献综述的真正功能

定量社科论文中的文献综述应建立“理论问题空间”，不是堆研究结果。

它应说明：

```text
已有理论如何解释该现象？
已有实证发现是否一致？
不一致可能来自什么机制或情境？
本文如何提出新的解释或检验？
```

### 8.2 文献综述组织方式

#### 方式一：理论流派式

```text
资源理论如何解释 X
身份理论如何解释 X
制度理论如何解释 X
本文整合或区分这些解释
```

#### 方式二：机制式

```text
已有研究关注直接效应
已有研究关注资源机制
已有研究关注认知机制
本文提出双机制或边界条件
```

#### 方式三：争议式

```text
研究发现 X 促进 Y
研究发现 X 抑制 Y
研究发现结果不一致
本文解释这种不一致
```

#### 方式四：方法 / 识别式

```text
早期横断面研究
后续纵向研究
准实验或实验研究
仍未解决的识别问题
本文设计如何改进
```

#### 方式五：测量式

```text
已有研究如何测量 X
测量局限
本文如何改进操作化
```

### 8.3 文献综述段落模板

```text
一类研究认为 X 通过 A 机制影响 Y。其理论基础是……，已有实证研究通常发现……。然而，这一路径难以解释……。另一类研究强调 B 机制，指出……。本文认为，两类解释并非互斥，而是取决于 Z 条件；在……情境下，A 机制更可能发挥作用，而在……情境下，B 机制更强。
```

### 8.4 常见错误

- 按作者逐篇罗列；
- 只总结已有显著关系；
- 不解释机制；
- 不处理不一致结果；
- 文献综述与假设脱节；
- 把经验发现当作理论；
- 引用很多但没有形成论证；
- 没有导出本文假设。

### Agent 质量判断

```text
1. 文献综述是否围绕理论机制组织？
2. 是否解释已有研究不一致？
3. 是否导出假设？
4. 是否处理竞争解释？
5. 是否与变量和模型对应？
6. 是否避免文献列表化？
```

## 9. 结果、分析或论证主体的组织方式

### 9.1 主体部分应围绕假设而不是表格

弱结构：

```text
表 1 描述统计
表 2 回归结果
表 3 稳健性
表 4 异质性
```

强结构：

```text
7.1 工作自主性与创新行为的主效应
7.2 心理授权的中介作用
7.3 任务不确定性的调节作用
7.4 异质性与边界条件
7.5 稳健性和替代解释
```

### 9.2 结果叙事的基本单位

每个结果段落应包含：

```text
1. 对应假设；
2. 使用的模型或表格；
3. 核心系数或效应；
4. 不确定性；
5. 效应量解释；
6. 是否支持假设；
7. 理论含义的简短过渡。
```

### 9.3 不支持假设的写法

不支持假设不是失败，而是理论修正机会。

弱写法：

```text
H2 未得到支持，不再讨论。
```

强写法：

```text
H2 未得到支持。该结果表明，心理授权可能并非工作自主性影响创新行为的主要路径。一个可能解释是，在本样本的高绩效压力环境中，自主性同时增加了责任负担，从而抵消了心理授权带来的积极作用。后续研究可进一步区分授权的资源属性和压力属性。
```

### 9.4 结果图的使用

定量社科论文常用图：

- 边际效应图；
- 交互作用图；
- 事件研究图；
- 系数图；
- 中介路径图；
- 异质性森林图；
- 分布图；
- 残差或诊断图；
- RDD 截断图；
- 平行趋势图。

图应服务解释，而不是美化。

### 9.5 表格与正文关系

正文不应逐项复述表格，而应回答：

```text
1. 哪个假设被检验？
2. 关键变量效应多大？
3. 结果是否稳定？
4. 理论上意味着什么？
5. 有没有例外或反常？
```

### 9.6 常见失败

| 失败模式 | 表现 | 修正 |
|---|---|---|
| 表格堆砌 | 大量回归表无叙事 | 按假设组织结果 |
| 显著性崇拜 | 只报星号 | 报效应量和 CI |
| 机制断裂 | 中介 / 调节与理论无关 | 回到机制推导 |
| 稳健性机械 | 换模型但不解释 | 针对威胁设计检验 |
| 不支持结果消失 | 不讨论反向或不显著 | 理论解释和边界讨论 |
| 控制变量喧宾夺主 | 讨论大量控制变量 | 聚焦核心关系 |

## 10. 讨论与结论写法

### 10.1 Discussion 的核心任务

Discussion 应把统计发现提升为理论贡献。

```text
统计发现：
X 与 Y 显著相关。

理论解释：
这种关系通过 M 机制发生，并受到 Z 情境影响。

理论贡献：
这说明已有理论关于 X 的单一正向假设需要被情境化。

实际意义：
实践者不应无条件推广 X，而应关注 Z 条件。
```

### 10.2 理论贡献表达公式

```text
本文不是简单证明 X 与 Y 相关，而是说明 X 通过 M 机制影响 Y，并且这一机制取决于 Z 条件。
```

```text
本文将 A 理论与 B 理论结合，解释了既有研究中关于 X-Y 关系不一致的发现。
```

```text
本文通过更强识别策略表明，既有相关性研究中观察到的关系并非完全由……解释。
```

```text
本文发现与 H2 不一致，提示现有理论可能忽略了……这一反向机制。
```

### 10.3 实际意义表达

实际意义应基于效应大小和研究情境，而非只基于显著性。

弱写法：

```text
结果对政策制定具有重要意义。
```

强写法：

```text
结果显示，政策实施后低收入家庭子女的升学概率提高约 3 个百分点。虽然效应幅度有限，但由于政策覆盖人群规模较大，其群体层面的潜在影响值得关注。
```

### 10.4 局限性写法

应具体、诚实但不自我否定。

常见局限：

- 数据横断面；
- 变量自陈；
- 样本代表性有限；
- 未测量混杂；
- 识别假设不可完全验证；
- 效应外推有限；
- 量表适用性；
- 数据时间滞后；
- 政策实施异质性；
- 机制测量不充分。

### 10.5 Conclusion 写法

结论应收束：

```text
问题
→ 理论机制
→ 数据与方法
→ 发现
→ 贡献
→ 边界
```

示例：

```text
本文研究了数字监控如何影响员工创新行为。基于自我决定理论，本文提出数字监控通过降低感知自主性抑制创新行为，并且该关系在高绩效压力情境下更强。利用三波员工—主管匹配数据，本文发现……。这些结果表明，数字化管理工具的行为后果取决于员工对控制和自主的解释。由于样本来自……行业，未来研究需要检验该机制在其他组织情境中的适用性。
```


审稿人会问：

```text
1. 作者是否声称因果？

5. 是否处理内生性？

```

审稿人会问：



## 12. 常见失败模式

### 12.1 数据分析有结果但理论贡献不清楚

表现：

- 回归显著；
- 表格完整；
- 但不知道理论上新在哪里。

修正：

```text
1. 回到理论争议；
2. 提炼机制或边界条件；
3. 将结果解释为支持、修正或挑战某理论；
4. 删除与理论无关的分析。
```

### 12.2 假设是经验猜测

表现：

```text
根据已有研究，X 可能影响 Y，因此提出 H1。
```

修正：

```text
1. 加入理论机制；
2. 说明为什么 X 会影响 Y；
3. 说明方向；
4. 处理反向可能。
```

### 12.3 变量操作化不匹配

表现：

- 理论说“社会资本”，变量却是“朋友数量”；
- 理论说“创新质量”，变量却是“专利数量”。

修正：

```text
1. 重新定义概念；
2. 解释代理变量合理性；
3. 增加替代测量；
4. 降低结论范围。
```

### 12.4 横断面数据强行因果

修正：

```text
1. 改用关联语言；
2. 增加时间顺序数据或滞后变量；
3. 做敏感性分析；
4. 在局限性中承认。
```

### 12.5 控制变量堆砌

表现：

- 控制所有可得变量；
- 没有理论依据；
- 可能控制了中介或碰撞变量。

修正：

```text
1. 依据理论或 DAG 选择控制变量；
2. 区分混杂、中介和结果后变量；
3. 报告不同控制集稳健性。
```

### 12.6 中介分析不成立

表现：

- X、M、Y 同时测量；
- 仍声称机制成立。

修正：

```text
1. 使用更谨慎语言；
2. 增加分时测量；
3. 承认中介因果假设；
4. 将结果写为与机制一致。
```

### 12.7 调节效应解释错误

表现：

- 只看交互项；
- 不画边际效应；
- 比较组内显著性。

修正：

```text
1. 报告边际效应；
2. 图示交互；
3. 检验组间差异；
4. 解释实际意义。
```

### 12.8 稳健性不针对威胁

表现：

- 换几个模型；
- 结果一致；
- 但不回应识别问题。

修正：

```text
1. 列出主要威胁；
2. 每个稳健性检验对应一个威胁；
3. 解释检验结果如何影响结论。
```

### 12.9 只讲显著性，不讲效应大小

修正：

```text
1. 报告标准化系数或边际效应；
2. 给出预测概率变化；
3. 与样本均值或现实单位比较；
4. 讨论实际意义。
```

## 13. Agent 写作辅助策略

### 13.1 Agent 总体流程

Agent 辅助定量社科论文时，应围绕“理论机制与经验识别之间的闭环”工作：

```text
1. 识别研究领域和论文类型；
2. 提取社会 / 行为问题；
3. 提取理论机制；
4. 检查假设是否由理论推出；
5. 检查变量是否操作化概念；
6. 检查数据和样本是否合适；
7. 检查模型和识别策略；
8. 检查结果是否对应假设；
9. 检查稳健性是否回应威胁；
10. 检查 Discussion 是否回到理论贡献；
11. 降级过度因果或泛化表述。
```

### 13.2 理论—假设—变量—模型—结果一致性检查

Agent 应生成如下矩阵：

| 理论机制 | 假设 | 变量 | 模型 | 结果 | 是否一致 |
|---|---|---|---|---|---|
| X 通过资源机制影响 Y | H1 | X, Y | 主效应模型 | β 显著 | 是 / 否 |
| M 传导 X-Y | H2 | M | 中介模型 | 间接效应 | 是 / 否 |
| Z 改变 X-Y | H3 | X×Z | 交互模型 | 边际效应 | 是 / 否 |

### 13.3 Agent 如何重构结果叙事

弱结果叙事：

```text
表 2 显示模型 1、模型 2、模型 3 的结果。X 的系数显著，M 的系数显著，交互项显著。
```

重构为：

```text
本文首先检验工作自主性对创新行为的主效应。结果显示……
随后检验心理授权的中介机制。结果表明……
最后检验任务不确定性的调节作用。边际效应图显示……
整体来看，结果支持本文关于“自主性—心理授权—创新行为”机制链条的核心预测。
```

### 13.4 Agent 如何重构贡献陈述

Agent 应从以下问题提炼贡献：

```text
1. 本文解释了哪个理论争议？
2. 本文发现了哪个机制？
3. 本文识别了哪个边界条件？
4. 本文是否使用更强数据或设计？
5. 本文是否修正已有理论假设？
6. 本文是否提供新的测量方式？
```

贡献句模板：

```text
本文的贡献不在于再次证明 X 与 Y 相关，而在于揭示 X 通过 M 机制影响 Y，并说明该机制仅在 Z 条件下成立，从而为……理论提供了边界化解释。
```

### 13.5 Agent 如何检查过度因果

高风险词：

```text
导致
影响
促进
抑制
提高
降低
决定
产生
因果作用
机制证明
```

判断：

```text
1. 是否随机实验？
2. 是否有准实验识别？
3. 是否有时间顺序？
4. 是否处理遗漏变量和反向因果？
5. 如果没有，应改为“相关”“关联”“预测”“与……一致”。
```

### 13.6 Agent 如何检查效应量解释

Agent 应提醒用户：

```text
1. 报告系数不够；
2. 解释单位变化；
3. 报告标准化效应或边际效应；
4. 给出实际情境中的变化；
5. 说明统计显著是否实际重要。
```

### 13.7 Agent 如何诊断稳健性

Agent 应先列威胁，再匹配检验：

| 威胁 | 当前是否处理 | 建议 |
|---|---|---|
| 共同方法偏差 | 否 | 多源数据 / Harman 检验不足时需谨慎 |
| 遗漏变量 | 部分 | 增加固定效应 / 敏感性分析 |
| 反向因果 | 否 | 滞后模型 / 面板 / 降低因果声称 |
| 极端值 | 否 | winsorize / 分位回归 |
| 替代测量 | 否 | 替代因变量 / 自变量 |
| 样本选择 | 否 | 权重 / 匹配 / 选择模型 |
